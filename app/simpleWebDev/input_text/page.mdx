# 글자 입력받기

<NyangSpeech>
  2부에 온 걸 환영한다냥! 지금까지는 우리가 화면에 글자를 **보여주기만**
  했다냥. 이제부터는 반대로, 사용자에게 **입력을 받아본다냥.** 파이썬의
  `input()` 기억나냥? Streamlit 버전 `input()` 을 배우는 거라냥!
</NyangSpeech>

---

## 오늘의 학습 목표

<Callout type="goal">
  - `st.text_input` 으로 **한 줄 글자**를 입력받는다.
  - `st.text_area` 로 **여러 줄 글자**를 입력받는다.
  - 입력받은 값을 **변수에 담아** 화면에 다시 보여준다.
</Callout>

---

## 1단계 · input 과 닮았다냥

<NyangSpeech>
  파이썬에서 이름을 물어볼 때 `name = input("이름? ")` 이라고 썼지?
  Streamlit 도 똑같다냥. `name = st.text_input("이름?")` — 함수만 바뀌고
  나머지는 그대로다냥. 입력한 값이 `name` 에 담긴다냥!
</NyangSpeech>

<Callout type="analogy">
  **input 과 text_input, 어디가 다르냥**

  | | 어디서 입력하냥 |
  |---|---|
  | `input("이름?")` | 까만 터미널에 타이핑 |
  | `st.text_input("이름?")` | 웹 화면의 입력창에 타이핑 |

  둘 다 **입력한 값을 돌려준다**는 점은 똑같다냥. 그래서 변수에 받아서
  쓰면 된다냥!
</Callout>

---

## 2단계 · 한 줄 입력 — text_input

```python
import streamlit as st

name = st.text_input("이름을 입력하세요")

st.write("안녕, " + name + " 님이다냥!")
```

<Callout type="note">
  **아직 아무것도 안 쳤을 때는?**

  입력창이 비어 있으면 `name` 은 **빈 문자열 `""`** 이라냥. 그래서 처음
  화면엔 "안녕,  님이다냥!" 처럼 이름 자리가 비어 보인다냥. 이걸 깔끔하게
  처리하는 법은 곧 배운다냥!
</Callout>

<Callout type="tip">
  **미리 채워두기 — value**

  `st.text_input("이름", value="코딩냥")` 처럼 `value` 를 주면 입력창에
  **기본값**이 미리 들어가 있다냥. 사용자가 지우고 새로 칠 수 있다냥.
</Callout>

---

## 3단계 · 여러 줄 입력 — text_area

<NyangSpeech>
  이름처럼 짧은 건 `text_input` 이 좋지만, 자기소개나 후기처럼 **긴 글**은
  `text_area` 가 좋다냥. 입력창이 넓은 상자로 바뀌어서 여러 줄을 쓸 수
  있다냥!
</NyangSpeech>

```python
import streamlit as st

story = st.text_area("자기소개를 써주세요")

st.write("입력한 소개는 아래와 같다냥:")
st.write(story)
```
