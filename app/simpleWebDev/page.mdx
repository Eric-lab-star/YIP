# Streamlit 웹 개발 입문

<NyangSpeech>

  다 배웠지? 그런데 지금까지 만든 건 전부 **까만 화면에 글자만** 나왔다냥.

## 이 수업에서 얻어갈 것

<Callout type="goal">
  - 파이썬 코드를 **웹 화면**으로 바꾸는 법
  - 웹사이트를 이루는 **구성요소**(화면 배치, 입력, 상태, 데이터)를 하나씩
  - 여러 페이지를 가진 **나만의 사이트**를 처음부터 끝까지 완성하기
  - 만든 사이트를 **인터넷에 배포하고 링크로 공유하기**
</Callout>

---

## 이렇게 배운다냥

<StepCard step="1" title="구성요소를 하나씩">
  Streamlit 함수를 외우는 수업이 아니라냥. **웹사이트가 무엇으로 이루어져
  있는지**를 화면 → 입력 → 기억 → 데이터 순서로 하나씩 얻어가는 수업이다냥.
</StepCard>

<StepCard step="2" title="만들면서 확인하기">
  중간중간 **미니 프로젝트**로 배운 걸 합쳐본다냥. BMI 계산기, 방명록,
  데이터 대시보드까지 작은 앱을 여러 개 완성하게 된다냥.
</StepCard>

<StepCard step="3" title="내 사이트로 마무리">
  마지막 6부에서는 **나만의 소개 사이트**를 기획부터 배포까지 직접 만든다냥.
  다 만들면 링크가 하나 생긴다냥. 그걸 친구한테 보내주면 된다냥!
</StepCard>

## 시작하기 전에

<Callout type="important">
  이 수업은 **파이썬 기본 문법을 아는 상태**를 전제로 한다냥.
  변수·조건문·반복문·리스트·딕셔너리가 낯설다면 **파이썬 탐험하기**를 먼저
  끝내고 오는 걸 추천한다냥!
</Callout>

<Callout type="note">
  **실습 환경 안내**

  Streamlit 앱은 브라우저 안에서 바로 돌릴 수 없다냥. 그래서 이 수업은 **내
  컴퓨터에 파이썬과 Streamlit을 직접 설치**하고 진행한다냥.

  설치가 처음이라도 걱정 말라냥 — **1강 전체**를 설치와 "화면이 안 떠요"
  해결에 쓴다냥!
</Callout>

## 수업 순서

이 수업은 여섯 개의 부, 모두 32강으로 되어 있다냥.

| 부 | 배우는 것 | 강 |
|---|---|---|
| 1부 · 첫 화면 만들기 | 설치, 글자 출력, 화면 배치 | 1~5강 |
| 2부 · 사용자에게 입력받기 | 입력 위젯, 재실행 규칙 | 6~11강 |

| 6부 · 나만의 사이트 | 기획부터 배포까지 | 29~32강 |
