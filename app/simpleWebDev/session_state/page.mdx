## 1단계 · 재실행에도 살아남는 상자

<NyangSpeech>
  일반 변수는 재실행 때마다 처음부터 다시 만들어진다냥. 그런데
  `st.session_state` 는 **재실행이 일어나도 그대로 남아 있는** 특별한
  창고다냥. 여기에 넣어둔 값은 안 사라진다냥!
</NyangSpeech>

<Callout type="analogy">
  **책상 vs 서랍**

  - 일반 변수 = **책상 위**. 재실행이라는 청소부가 올 때마다 싹 치운다냥.
  - `session_state` = **서랍 안**. 청소부가 와도 서랍은 안 건드린다냥.

  기억하고 싶은 값은 서랍(`session_state`)에 넣으면 된다냥!
</Callout>

---

## 2단계 · 저장하고 꺼내기

<NyangSpeech>
  `session_state` 는 **딕셔너리처럼** 쓴다냥. 파이썬 탐험하기에서 배운
  딕셔너리 기억나냥? 열쇠(key)로 값을 넣고 꺼내는 거 말이다냥.
</NyangSpeech>

```python
import streamlit as st

# 저장하기 — 딕셔너리처럼 열쇠에 값을 넣는다냥
st.session_state["nickname"] = "코딩냥"

# 꺼내기
st.write(st.session_state["nickname"])
```

<Callout type="important">
  **없는 값을 꺼내면 에러가 난다냥**

  서랍에 아직 안 넣은 열쇠를 꺼내려 하면 에러가 난다냥. 그래서 보통
  **처음 한 번만 초기화**하는 코드를 맨 위에 둔다냥.

  ```python
  if "count" not in st.session_state:
      st.session_state["count"] = 0
  ```

  "이미 있으면 그대로 두고, 없을 때만 0으로 시작" 이라는 뜻이라냥.
</Callout>

## 3단계 · 드디어 카운터 완성

<NyangSpeech>
  자, 9강의 원수를 갚을 시간이다냥! 이번엔 진짜로 숫자가 쌓이는 카운터를
  만들어보자냥.
</NyangSpeech>

```python
import streamlit as st

# 1) 처음 한 번만 0으로 초기화
if "count" not in st.session_state:
    st.session_state["count"] = 0

# 2) 버튼을 누르면 서랍 속 값을 1 늘린다냥
if st.button("증가"):
    st.session_state["count"] = st.session_state["count"] + 1

# 3) 서랍 속 값을 보여준다냥
st.write("횟수: " + str(st.session_state["count"]))
```

<Callout type="tip">
  **9강과 무엇이 다르냥**

  달라진 건 딱 하나 — `count` 를 **일반 변수 대신 `session_state` 에**
  넣은 거다냥. 이제 재실행이 일어나도 서랍 속 값은 안 사라지니까, 누를
  때마다 1, 2, 3, ... 차곡차곡 쌓인다냥!
</Callout>

## 4단계 · 직접 해보기

<Callout type="analogy">
  **미션 · 좋아요 버튼**

  - "좋아요" 버튼을 만들고
  - 누를 때마다 `session_state` 의 숫자를 1 늘리고
  - "지금까지 좋아요 OO개" 를 보여주자냥.

  카운터와 거의 같다냥. 초기화 → 버튼으로 증가 → 보여주기, 이 세 박자를
  손에 익히는 게 목표라냥!
</Callout>
