## 1단계 · 빈 화면을 친절하게

<NyangSpeech>
  방명록에 아직 글이 하나도 없으면 화면이 휑하다냥. "아직 글이 없다냥,
  첫 글을 남겨보라냥!" 같은 안내를 넣으면 훨씬 친절하다냥. 이걸 **빈 상태
  처리** 라고 한다냥.
</NyangSpeech>

```python
st.subheader("남겨진 글")

if len(st.session_state["guestbook"]) == 0:
    st.info("아직 글이 없다냥. 첫 방문자가 되어보라냥!")
else:
    for line in st.session_state["guestbook"]:
        st.write("- " + line)
```

<Callout type="tip">
  **빈 상태를 챙기는 습관**

  목록이 비었을 때, 검색 결과가 없을 때, 파일을 안 올렸을 때 —
  이런 "아무것도 없는 순간" 에 안내를 넣는 게 좋은 앱의 조건이라냥.
  `if len(...) == 0:` 으로 확인하면 된다냥!
</Callout>

## 2단계 · 테마로 색 입히기

<NyangSpeech>
  28강에서 배운 테마를 적용해보자냥. `.streamlit/config.toml` 을 만들어
  내 사이트만의 색을 정하는 거다냥. 노란 고양이색이 잘 어울린다냥!
</NyangSpeech>

```text
# .streamlit/config.toml

[theme]
primaryColor = "#F59E0B"
backgroundColor = "#FFFDF7"
textColor = "#1F2937"
```

<Callout type="note">
  설정 파일을 만든 뒤 앱을 새로 실행하면 색이 바뀐다냥. 버튼과 강조가
  노란색으로, 배경이 살짝 크림색으로 변한다냥. 훨씬 아늑해진다냥!
</Callout>

@st.cache_data
def load_hobby():
