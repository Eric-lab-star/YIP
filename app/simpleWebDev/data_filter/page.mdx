## 오늘의 학습 목표

<Callout type="goal">
  - 조건으로 표의 **행을 걸러낸다(필터).**
  - 슬라이더·선택 위젯을 표와 **연결**한다.
  - 위젯을 움직이면 표가 **바로 바뀌는** 앱을 만든다.

## 1단계 · 조건으로 행 걸러내기

<NyangSpeech>
  pandas 에서 조건을 대괄호 안에 넣으면, 그 조건에 **맞는 행만** 남는다냥.
  예를 들어 "나이가 3 이하" 인 냥이만 골라보자냥.
</NyangSpeech>

```python
import streamlit as st
import pandas as pd

df = pd.read_csv("cats.csv")

young = df[df["나이"] <= 3]      # 나이가 3 이하인 행만 남긴다냥

st.dataframe(young)
```

<Callout type="note">
  **`df[조건]` 의 뜻**

  `df["나이"] <= 3` 은 각 행마다 참/거짓을 만든다냥. 그걸 `df[...]` 에
  넣으면 **참인 행만** 남는다냥. "조건에 맞는 것만 남겨라" 라고 읽으면
  된다냥!
</Callout>

## 2단계 · 슬라이더와 표 연결하기

<NyangSpeech>
  이제 조건의 숫자(3)를 슬라이더로 바꿔보자냥. 9강의 재실행 규칙 덕분에,
  슬라이더를 움직일 때마다 표가 저절로 다시 걸러진다냥!
</NyangSpeech>

```python
import streamlit as st
import pandas as pd

df = pd.read_csv("cats.csv")

max_age = st.slider("최대 나이", 0, 10, 5)

result = df[df["나이"] <= max_age]

st.write("나이 " + str(max_age) + "살 이하 냥이")
st.dataframe(result)
```

<Callout type="tip">
  **이게 바로 대화형 앱이라냥**

  슬라이더를 밀면 → 재실행 → 새 조건으로 표가 걸러짐 → 화면 갱신. 이
  흐름이 Streamlit 의 진짜 매력이라냥. 코드는 몇 줄인데 진짜 검색 도구처럼
  동작한다냥!
</Callout>

## 3단계 · 글자로 검색하기

<NyangSpeech>
  숫자뿐 아니라 **글자로도** 검색할 수 있다냥. 입력한 글자가 들어간 이름만
  찾아보자냥.
</NyangSpeech>

```python
keyword = st.text_input("이름 검색")

if keyword:
    found = df[df["이름"].str.contains(keyword)]
    st.dataframe(found)
else:
    st.dataframe(df)      # 검색어가 없으면 전체를 보여준다냥
```

<Callout type="note">
  **`.str.contains(...)` 는 "포함하니?" 라냥**

  `df["이름"].str.contains("냥")` 은 이름에 "냥" 이 들어간 행을 찾는다냥.
  검색창이 비었을 땐(`if keyword:` 가 거짓) 전체를 보여주는 게 자연스럽다냥.
</Callout>
