## 오늘의 학습 목표

<Callout type="goal">
  - `st.image` 로 **사진**을 화면에 띄운다.
  - `st.file_uploader` 로 사용자가 **파일을 올리게** 한다.
  - 올린 이미지·CSV 를 **받아서 처리**한다.
</Callout>

## 1단계 · 이미지 띄우기 — image

<NyangSpeech>
  `st.image` 는 사진을 화면에 보여준다냥. 인터넷 주소를 주거나, 내 컴퓨터의
  파일 경로를 주면 된다냥.
</NyangSpeech>

```python
import streamlit as st

st.image("https://placekitten.com/400/300", caption="귀여운 냥이라냥")
```

<Callout type="tip">
  **`caption` 은 사진 설명이라냥**

  `caption="..."` 을 주면 사진 아래에 작은 설명이 붙는다냥. `width=300`
  으로 크기도 정할 수 있다냥!
</Callout>

---

## 2단계 · 파일 올리게 하기 — file_uploader

<NyangSpeech>
  `st.file_uploader` 는 사용자가 **자기 파일을 올릴 수 있는** 상자를
  만든다냥. 올리기 전엔 `None`(아무것도 없음)이고, 올리면 그 파일이
  담긴다냥.
</NyangSpeech>

```python
import streamlit as st

uploaded = st.file_uploader("이미지를 올려주세요")

if uploaded is not None:
    st.image(uploaded, caption="올린 사진이다냥")
else:
    st.info("아직 올린 파일이 없다냥")
```

<Callout type="important">
  **올리기 전에는 None 이라냥**

  파일을 올리기 전에 `uploaded` 는 `None` 이다냥. 그래서
  `if uploaded is not None:` 으로 **올렸을 때만** 처리해야 에러가 안
  난다냥. 6강에서 배운 "빈 입력 확인" 과 같은 맥락이라냥!
</Callout>

---

## 3단계 · 올린 CSV 읽기

<NyangSpeech>
  이미지뿐 아니라 CSV 파일도 올려서 바로 표로 볼 수 있다냥. 18강의
  `read_csv` 를 올린 파일에 그대로 쓰면 된다냥!
</NyangSpeech>

```python
import streamlit as st
import pandas as pd

uploaded = st.file_uploader("CSV 파일을 올려주세요", type="csv")

if uploaded is not None:
    df = pd.read_csv(uploaded)
    st.dataframe(df)
```

<Callout type="note">
  **`type="csv"` 로 파일 종류 제한**

  `type="csv"` 를 주면 CSV 파일만 올릴 수 있게 막아준다냥. 이미지면
  `type=["png", "jpg"]` 처럼 리스트로 여러 개도 된다냥!
</Callout>
