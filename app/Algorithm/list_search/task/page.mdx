## 📥 먼저 · 입력 받는 법

<NyangSpeech>
  개념 페이지에선 리스트를 코드 안에 직접 적어뒀지만, 채점 문제는 **입력으로**
  받는다냥. 이 두 줄만 외워두면 된다냥! 🐱
</NyangSpeech>

```python
n = int(input())                            # 숫자가 몇 개인지
numbers = list(map(int, input().split()))   # 숫자들을 리스트로
```

<StepCard step="해설" title="두 번째 줄이 하는 일">
  `input()` 은 `"3 9 1 7 5"` 같은 **문자열 한 줄**을 받는다냥.

  - `.split()` → `["3", "9", "1", "7", "5"]` 로 쪼갠다냥
  - `map(int, ...)` → 각 조각을 **숫자로** 바꾼다냥
  - `list(...)` → 리스트로 묶는다냥

  결과는 `[3, 9, 1, 7, 5]` 라냥! 🐾
</StepCard>
