<MissionCard number={1} title="1부터 n까지의 합" time="7분">
  n 을 입력받아 **1부터 n까지 모두 더한 값**을 출력하는 문제다냥.

  [문제 풀러 가기](/problems/s-sum-1-to-n)

  <Callout type="important">
    `range(n)` 이 아니라 `range(1, n + 1)` 이라냥. `+1` 을 빠뜨리면 n 이
    빠져서 답이 딱 n 만큼 모자라게 나온다냥.

    답이 조금 작게 나오면 이걸 제일 먼저 의심하자냥.
  </Callout>

  ```python
  n = int(input())

  total = 0
  for i in range(1, n + 1):
      # 여기서 total 에 더한다냥
      pass

  print(total)
  ```
</MissionCard>

<MissionCard number={2} title="팩토리얼" time="7분">
  n 을 입력받아 **1부터 n까지 모두 곱한 값**을 출력하는 문제다냥.

  [문제 풀러 가기](/problems/s-factorial)

  <Callout type="important">
    1번과 뼈대가 똑같다냥. 딱 두 군데만 다르다냥.

    - 상자 시작값: `0` 이 아니라 **`1`**
    - 갱신: `+` 가 아니라 **`*`**

    시작값을 0으로 두면 답이 무조건 0이 나온다냥.
  </Callout>

  ```python
  n = int(input())

  result = 1
  for i in range(1, n + 1):
      # 여기서 result 에 곱한다냥
      pass

  print(result)
  ```
</MissionCard>

<MissionCard number={3} title="카운트다운" time="7분">
  n 부터 1까지 **거꾸로** 출력하는 문제다냥.

  [문제 풀러 가기](/problems/s-countdown)

  <Callout type="tip">
    `range` 의 세 번째 자리에 **음수**를 넣으면 거꾸로 간다냥.

    `range(n, 0, -1)` 은 n부터 시작해서 0 직전까지, 1씩 빼면서 간다냥.
    0 직전까지니까 마지막은 1 이라냥.

    출력 형식을 문제에서 확인하자냥. 한 줄씩인지 한 줄에 모아서인지에 따라
    `print` 쓰는 법이 달라진다냥.
  </Callout>

  ```python
  n = int(input())

  for i in range(n, 0, -1):
      pass
  ```
</MissionCard>

<MissionCard number={4} title="각 자리 숫자의 합" time="12분">
  숫자를 입력받아 **각 자리 숫자를 모두 더한 값**을 출력하는 문제다냥.
  `1234` 면 `1+2+3+4` 라서 `10` 이라냥.

  [문제 풀러 가기](/problems/s-sum-digits)

  <Callout type="tip">
    개념 페이지의 **사탕 꺼내기**를 그대로 쓰면 된다냥.

    - `n % 10` 으로 맨 뒤 자리를 꺼내 더하고
    - `n // 10` 으로 그 자리를 버린다냥
    - `n` 이 0이 될 때까지 `while` 로 반복한다냥

    문자열로 바꿔서 푸는 방법도 있다냥. 먼저 `while` 로 풀어보고, 다 풀었으면
    문자열 방법으로도 한 번 짜보면 좋다냥. 두 방법을 다 아는 게 중요하다냥.
  </Callout>

  ```python
  n = int(input())

  total = 0
  while n > 0:
      # 맨 뒤 자리를 더하고, 그 자리를 버린다냥
      pass

  print(total)
  ```
</MissionCard>

<MissionCard number={5} title="n번째 피보나치 수" time="15분">
  오늘의 마지막이자 제일 어려운 문제다냥. **앞의 두 수를 더해 만든 수열**의
  n번째 값을 구한다냥.

  [문제 풀러 가기](/problems/s-fib-nth)

  <Callout type="important">
    상자가 **두 개** 필요하다냥. 그리고 갱신은 반드시 **한 줄로** 해야 한다냥.

    ```python
    a, b = b, a + b     # 맞다냥
    ```

    두 줄로 쪼개면 `a` 가 먼저 바뀌어서 틀린 답이 나온다냥.
  </Callout>

  <StepCard step="주의" title="시작값과 횟수를 꼭 확인하자냥">
    피보나치는 문제마다 **1번째가 무엇인지**가 다르다냥. 어떤 데선 `1, 1` 로
    시작하고 어떤 데선 `0, 1` 로 시작한다냥.

    문제의 **예시 입력과 출력**을 보고 맞춰야 한다냥. 로직이 맞아도 시작값이나
    반복 횟수가 한 칸 어긋나면 오답이 된다냥. 예시로 손 계산을 꼭 해보자냥.
  </StepCard>

  ```python
  n = int(input())

  a, b = 1, 1
  # 몇 번 돌아야 하냥? 예시로 확인하고 정하자냥

  print(a)
  ```
</MissionCard>
