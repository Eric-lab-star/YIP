<MissionCard number={1} title="오름차순 정렬" time="7분">
  정수들을 **작은 것부터** 정렬해 한 줄에 공백으로 구분해 출력하는 문제다냥.

  [문제 풀러 가기](/problems/s-sort-asc)

  <Callout type="tip">
    `sorted()` 를 쓰면 정렬은 끝이라냥. 남은 건 **출력 형식**이라냥.

    리스트를 그냥 `print()` 하면 `[1, 2, 3]` 처럼 대괄호와 쉼표가 같이 나와서
    오답이 된다냥. 공백으로 구분해 출력하려면 이렇게 쓴다냥.

    ```python
    print(*arr)
    ```

    `*` 는 리스트를 풀어헤쳐서 하나씩 넘긴다는 뜻이라냥.
  </Callout>

  ```python
  n = int(input())
  arr = list(map(int, input().split()))

  # 정렬해서 공백 구분으로 출력한다냥
  ```
</MissionCard>

<MissionCard number={2} title="상위 k개" time="10분">
  **큰 값부터 k개**를 출력하는 문제다냥.

  [문제 풀러 가기](/problems/s-top-k)

  <Callout type="tip">
    두 가지를 붙이면 된다냥 — 큰 것부터 정렬(`reverse=True`)하고, 앞에서 k개를
    슬라이싱한다냥.

    첫 줄에 `n` 과 `k` 가 **함께** 주어진다는 걸 놓치지 말자냥.
  </Callout>

  ```python
  n, k = map(int, input().split())
  arr = list(map(int, input().split()))

  # 큰 것부터 k개를 공백 구분으로 출력한다냥
  ```
</MissionCard>

<MissionCard number={3} title="중앙값" time="10분">
  정렬했을 때 **정확히 가운데 오는 값**을 출력하는 문제다냥. `n` 은 항상
  홀수라냥.

  [문제 풀러 가기](/problems/s-median)

  <Callout type="important">
    가운데 인덱스는 `n // 2` 라냥. 슬래시 **두 개**라냥.

    `n / 2` 를 쓰면 소수가 나와서 인덱스로 못 쓰고 에러가 난다냥.
  </Callout>

  <StepCard step="확인" title="직접 세어보자냥">
    `n` 이 5면 인덱스는 0 1 **2** 3 4 라서 가운데는 2번이고, `5 // 2` 가 2라냥.

    `n` 이 7이면? 0 1 2 **3** 4 5 6 이라서 3번이고 `7 // 2` 가 3이라냥. 맞다냥.
  </StepCard>

  ```python
  n = int(input())
  arr = list(map(int, input().split()))

  # 정렬한 뒤 가운데 값을 출력한다냥
  ```
</MissionCard>

<MissionCard number={4} title="두 번째로 큰 수" time="12분">
  **서로 다른 값 중에서** 두 번째로 큰 값을 출력하는 문제다냥.

  [문제 풀러 가기](/problems/s-second-largest)

  <Callout type="important">
    오늘 배운 함정이 그대로 나오는 문제라냥.

    `5 5 3` 이 주어지면 답은 뭐냥? 그냥 정렬해서 뒤에서 두 번째를 꺼내면
    **5** 가 나오는데, 문제가 원하는 답은 **3** 이라냥.

    "서로 다른 값 중에서"라고 했으니 **중복을 먼저 없애야** 한다냥.
  </Callout>

  <Callout type="tip">
    `set(arr)` 을 쓰면 중복이 사라진다냥. 다만 `set` 은 순서가 없으니 그
    다음에 다시 정렬해야 한다냥.

    예시 입력 중에 중복이 들어있는 게 있다냥. **실행 버튼으로 꼭 확인하고**
    제출하자냥.
  </Callout>

  ```python
  n = int(input())
  arr = list(map(int, input().split()))

  # 중복을 없애고, 정렬하고, 뒤에서 두 번째를 꺼낸다냥
  ```
</MissionCard>

<MissionCard number={5} title="길이순 정렬" time="15분">
  오늘의 마지막이자 제일 어려운 문제다냥. 단어를 **길이가 짧은 것부터**,
  길이가 같으면 **사전순**으로 출력한다냥.

  [문제 풀러 가기](/problems/s-sort-by-length)

  <Callout type="tip">
    **튜플 키**를 쓸 차례라냥.

    ```python
    key=lambda s: (len(s), s)
    ```

    앞의 `len(s)` 로 먼저 비교하고, 같을 때만 뒤의 `s` 로 비교한다냥.

    `key=len` 만 쓰면 길이는 맞는데 같은 길이끼리 순서가 보장되지 않아 틀린다냥.
  </Callout>

  <StepCard step="주의" title="입력 읽는 법이 다르다냥">
    이 문제는 숫자가 아니라 **여러 줄의 단어**가 들어온다냥.

    ```python
    n = int(input())
    words = [input().strip() for _ in range(n)]
    ```

    `.strip()` 을 빼먹으면 줄바꿈이 단어에 붙어서 길이가 1씩 크게 나온다냥.
    그러면 정렬 순서가 통째로 틀어진다냥.
  </StepCard>

  ```python
  n = int(input())
  words = [input().strip() for _ in range(n)]

  # 길이순, 같으면 사전순으로 한 줄에 하나씩 출력한다냥
  ```
</MissionCard>
