<MissionCard number={2} title="문장을 벡터로 변환하기 🔢" time="8~10분">

미리 준비된 여러 문장을 **임베딩 API**로 숫자 벡터로 바꿔볼 거다냥. 벡터가 몇 개의 숫자로 이루어졌는지(차원 수)도 확인해보자냥!

```python
import google.generativeai as genai

genai.configure(api_key="여기에 본인의 제미나이 API 키 입력")

sentences = [
    "오늘 날씨가 정말 좋다.",
    "오늘 기분이 너무 화창해.",
    "나는 점심으로 김치볶음밥을 먹었다.",
]

embeddings = []
for sentence in sentences:
    result = genai.embed_content(
        model="models/text-embedding-004",
        content=sentence
    )
    embeddings.append(result["embedding"])

print("벡터의 길이(차원 수):", len(embeddings[0]))
print("첫 번째 문장 벡터 일부:", embeddings[0][:5])
```

> ✅ **확인해보자냥**
> - 벡터의 길이(차원 수)가 숫자로 출력됐냥?
> - 벡터 일부를 보면 숫자가 주르륵 나오지냥? 사람이 봐서는 무슨 의미인지 모르겠지냥? 🐾

<Callout type="tip">
  💡 **벡터 자체는 사람이 읽어도 의미를 알 수 없다냥!**

  숫자가 잔뜩 나열된 벡터는 **컴퓨터를 위한 표현 방식**이라냥. 우리는 의미를 모르지만, 컴퓨터는 이 숫자들로 문장끼리 얼마나 비슷한지 계산할 수 있다냥!
</Callout>

</MissionCard>

이제 두 문장이 얼마나 비슷한지 **코사인 유사도**로 계산해볼 거다냥. 아래 템플릿에서 **빈칸(`____`)을 직접 채워보자냥.**

```python
import numpy as np

def cosine_similarity(vec1, vec2):
    vec1 = np.array(vec1)
    vec2 = np.array(vec2)
    return np.dot(vec1, vec2) / (np.linalg.norm(vec1) * np.linalg.norm(vec2))

# 문장 0번과 1번의 유사도 (의미가 비슷한 문장끼리)
sim_0_1 = cosine_similarity(embeddings[0], embeddings[1])

# 문장 0번과 2번의 유사도 (의미가 다른 문장끼리) - 직접 계산해보세요
sim_0_2 = ____

print(f"'날씨가 좋다' vs '기분이 화창해' 유사도: {sim_0_1:.4f}")
print(f"'날씨가 좋다' vs '점심을 먹었다' 유사도: {sim_0_2:.4f}")
```

> ✅ **확인해보자냥 (빈칸 힌트)**
> - `sim_0_2 = ____` 자리에는 위의 `cosine_similarity` 함수를 똑같이 써서, 이번엔 `embeddings[0]` 과 `embeddings[2]` 를 넣어주면 된다냥!
> - 의미가 비슷한 0번-1번이, 의미가 다른 0번-2번보다 유사도가 더 **높게** 나오냥? 🐱

<Callout type="important">
  ⚠️ **유사도 값이 직관과 다르게 나올 수도 있다냥!**

  AI도 완벽하지 않다냥. 유사도는 정답이 아니라 **참고용 점수**라고 생각하자냥. 대체로 의미가 비슷하면 점수가 높게 나오지만, 가끔 예상과 다를 수도 있다냥!
</Callout>

<MissionCard number={4} title="나만의 문장으로 실험하기 ✍️" time="5~7분">

이번엔 **내가 직접** 문장을 만들어서 실험해볼 거다냥! 의미가 **비슷한 문장 2개**, 의미가 **다른 문장 1개**를 만들어서 유사도를 비교해보자냥.

```python
# 의미가 비슷한 문장 2개, 의미가 다른 문장 1개를 직접 작성해보세요
my_sentences = [
    ____,  # 문장 A
    ____,  # 문장 B (A와 의미가 비슷하게)
    ____,  # 문장 C (A, B와 의미가 다르게)
]

my_embeddings = []
for sentence in my_sentences:
    result = genai.embed_content(model="models/text-embedding-004", content=sentence)
    my_embeddings.append(result["embedding"])

print("A-B 유사도:", cosine_similarity(my_embeddings[0], my_embeddings[1]))
print("A-C 유사도:", cosine_similarity(my_embeddings[0], my_embeddings[2]))
```

> ✅ **확인해보자냥 (빈칸 힌트)**
> - 빈칸에는 큰따옴표로 감싼 문장을 넣으면 된다냥! 예: `"나는 강아지를 좋아해."`
> - 문장 A와 B는 의미가 비슷하게 (예: "강아지가 귀엽다" / "개가 너무 사랑스러워")
> - 문장 C는 전혀 다른 주제로 (예: "오늘 수학 숙제가 많다")

✍️ **직접 해보기**
A-B 유사도가 A-C 유사도보다 **높게** 나오는지 확인해보자냥. 의미가 비슷하면 가까운 좌표에 놓인다는 걸 내 눈으로 확인하는 거다냥! 🐟

## 🗺️ 핵심 비유 — 단어 지도 위의 좌표

<StepCard step="비유" title="유사도는 두 좌표 사이의 거리다냥!">

임베딩으로 변환된 문장들은 **거대한 지도 위의 좌표**와 같다냥.

- **"강아지"** 와 **"개"** 는 지도에서 아주 **가까운 동네**에 있다냥.
- **"강아지"** 와 **"자동차"** 는 지도에서 **멀리 떨어진 동네**에 있다냥.

유사도 계산은 바로 **"두 좌표 사이의 거리를 재는 것"** 이다냥. 거리가 가까울수록(유사도가 높을수록) 의미가 비슷하다는 뜻이라냥! 🧭
