```python
from google import genai
import numpy as np

client = genai.Client(api_key="여기에 본인의 제미나이 API 키 입력")

def get_embedding(text):
    result = client.models.embed_content(model="gemini-embedding-001", contents=text)
    return result.embeddings[0].values

def cosine_similarity(vec1, vec2):
    vec1, vec2 = np.array(vec1), np.array(vec2)
    return np.dot(vec1, vec2) / (np.linalg.norm(vec1) * np.linalg.norm(vec2))

# 모든 청크를 임베딩으로 변환
chunk_embeddings = [get_embedding(chunk) for chunk in chunks]

def search(query, top_k=1):
    query_embedding = get_embedding(query)

    # 각 청크와 질문 사이의 유사도를 계산해보세요
    similarities = []
    for chunk_emb in chunk_embeddings:
        sim = ____  # cosine_similarity 함수를 사용해보세요
        similarities.append(sim)

    # 유사도가 가장 높은 청크의 인덱스(top_k개) 찾기
    top_indices = np.argsort(similarities)[::-1][:top_k]

    return [chunks[i] for i in top_indices]
```

> **확인해보자냥 (빈칸 힌트)**
> - `sim = ____` 자리에는 `cosine_similarity` 함수에 **질문 임베딩(`query_embedding`)**과 **청크 임베딩(`chunk_emb`)** 두 개를 넣으면 된다냥.
> - `question = ____` 자리에는 문서 내용과 관련된 질문을 **따옴표로 감싼 문자열**로 넣으면 된다냥. (예: `"동아리는 어떻게 가입하나요?"`)

<Callout type="analogy">
  **비유로 이해하기 — 도서관 사서에게 질문하기** ‍

  오늘 만든 `search` 함수는 **도서관 사서**와 같다냥.

  - 손님(질문)이 와서 "이런 내용 찾고 있어요"라고 말하면,
  - 사서는 책장에 꽂힌 **모든 챕터(청크)의 위치(임베딩)**를 살펴보고,
  - 질문과 **가장 가까운 위치**에 있는 챕터를 꺼내 건네준다냥.

  아직 그 내용을 "설명"해주지는 않지만, **정확한 자료를 찾아주는 역할**까지는 완성한 거다냥!
