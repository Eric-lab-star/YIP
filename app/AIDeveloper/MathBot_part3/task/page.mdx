<MissionCard number={2} title="사진 핸들러 추가하기 " time="12~15분">

이제 오늘의 핵심이다냥! 아래 템플릿에서 **빈칸(`____`)을 직접 채워보자냥.** 사진을 받아 저장하고, 풀이를 만들어 답장하는 핸들러를 완성하는 거다냥!

```python
from telegram import Update
from telegram.ext import Application, MessageHandler, CommandHandler, filters, ContextTypes
from google import genai
import PIL.Image

TELEGRAM_TOKEN = "여기에 봇 토큰 입력"
client = genai.Client(api_key="여기에 제미나이 API 키 입력")

# 17차시에서 만든 풀이 프롬프트 재사용
solve_prompt = """
너는 친절한 수학 선생님이야. 이 사진 속 수학 문제를 풀어줘.
문제를 설명하고, 단계별 풀이 과정을 보여준 뒤, 마지막에 '정답: '으로 정답을 알려줘.
중학생도 이해할 수 있게 쉽게 설명해줘.
"""

async def start(update: Update, context: ContextTypes.DEFAULT_TYPE):
    await update.message.reply_text("수학 문제 사진을 보내주시면 풀이를 알려드릴게요!")

# 사진을 받았을 때 처리하는 함수
async def handle_photo(update: Update, context: ContextTypes.DEFAULT_TYPE):
    # 1) 사용자에게 처리 중임을 알리기
    await update.message.reply_text("문제를 푸는 중이에요. 잠시만 기다려주세요...")

    # 2) 사진 파일 받아서 저장
    photo_file = await update.message.photo[-1].get_file()
    await photo_file.download_to_drive("received.jpg")

    # 3) 저장된 사진으로 풀이 생성 - 직접 작성해보세요
    image = PIL.Image.open("received.jpg")
    response = client.models.generate_content(model="gemini-3.5-flash", contents=[____, image])  # 풀이 프롬프트와 이미지 전달

    # 4) 풀이 결과를 답장으로 보내기
    await update.message.reply_text(____)

app = Application.builder().token(TELEGRAM_TOKEN).build()
app.add_handler(CommandHandler("start", start))
app.add_handler(MessageHandler(filters.PHOTO, handle_photo))

print("수학 풀이 봇이 실행 중입니다...")
app.run_polling()
```

> **확인해보자냥 (빈칸 힌트)**
> - 첫 번째 `____`에는 AI에게 줄 **풀이 명령서**가 들어가야 한다냥. 위에서 만든 변수 이름을 떠올려보자냥! (`s`로 시작한다냥 )
> - 두 번째 `____`에는 AI가 만들어준 **풀이 결과 텍스트**가 들어가야 한다냥. `response`에서 텍스트만 꺼내려면 무엇을 붙여야 할까냥?

<Callout type="tip">
  **`photo[-1]`은 무슨 뜻이냥?**

  텔레그램은 사진 한 장을 보낼 때 여러 가지 크기로 함께 보낸다냥. `photo[-1]`은
  그중에서 **가장 마지막(가장 고화질)** 사진을 골라낸다는 뜻이다냥! 문제를
  또렷하게 읽으려면 고화질이 좋겠지냥?
</Callout>

### 자주 나는 오류들

<Callout type="important">
  **봇이 사진에 반응하지 않거나 오류가 난다면?**

  - **핸들러 등록 누락:** `app.add_handler(MessageHandler(filters.PHOTO, handle_photo))` 줄을 빠뜨리지 않았는지 확인하자냥. 이게 없으면 봇이 사진을 무시한다냥!
  - **토큰/키 혼동:** 봇 토큰 자리에 제미나이 키를, 제미나이 키 자리에 봇 토큰을 넣지 않았는지 다시 보자냥.
  - **들여쓰기 오류:** 파이썬은 들여쓰기에 아주 예민하다냥. `async def` 안쪽 줄들이 잘 맞춰져 있는지 확인하자냥!
</Callout>

</MissionCard>
