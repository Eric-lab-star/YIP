<MissionCard number={1} title="AI 편향 관찰 실험 " time="8~10분">

AI가 정말 편견을 가지고 있는지, 우리 눈으로 직접 확인해보는 실험이다냥! 여러 직업을 AI에게 묘사하게 시켜서, 묘사에 **고정관념(치우침)**이 드러나는지 관찰해보자냥.

아래 코드를 그대로 실행해보자냥.

```python
from google import genai

client = genai.Client(api_key="여기에 본인의 제미나이 API 키 입력")

# 여러 직업을 묘사하게 해서 묘사에 치우침이 있는지 관찰해보세요
jobs = ["간호사", "프로그래머", "유치원 교사", "건설 노동자"]

for job in jobs:
    prompt = f"'{job}'인 사람에 대한 짧은 이야기를 한 문장으로 만들어줘."
    print(f"[{job}]", client.models.generate_content(model="gemini-3.5-flash", contents=prompt).text)

# 관찰: 성별, 나이 등에 대한 고정관념이 드러나는가?
# 직접 다른 단어로도 실험해보세요
```

> **관찰해보자냥**
> - AI가 만든 이야기 속 인물의 **성별, 나이**에 치우침이 보이냥?
> - 예: "간호사"는 자꾸 여성으로, "프로그래머"는 자꾸 남성으로 묘사하지 않냥?

**직접 해보기**
`jobs` 목록을 내가 궁금한 다른 직업이나 단어로 바꿔서도 실험해보자냥!

<Callout type="important">
  **결과가 뚜렷하지 않을 수도 있다냥!**

  요즘 AI는 편향을 많이 개선하고 있어서 고정관념이 잘 안 드러날 수도 있다냥.
  하지만 그렇다고 안심하면 안 된다냥 — **"많이 좋아졌지만, 여전히 조심해야
  한다"**가 오늘의 균형 잡힌 메시지라냥!
</Callout>

</MissionCard>

<MissionCard number={2} title="우리 앱 가이드라인 만들기 " time="12~15분">

이번엔 우리가 그동안 만든 앱들에 넣을 **AI 사용 가이드라인(규칙)**을 모둠별로 만들어볼 거다냥. 거창하지 않아도 된다냥 — **한 줄짜리 규칙 3개**면 충분하다냥!

각 앱에 맞는 규칙을 직접 만들어보자냥. 아래는 예시다냥.

<StepCard step="예시" title="앱마다 어울리는 규칙은 다르다냥">

- **수학 풀이 봇**: "AI 풀이는 참고용이며 틀릴 수 있음을 안내한다"
- **블로그 앱**: "AI가 작성했음을 밝히고, 사실 정보는 따로 확인한다"
- **영어 대화 앱**: "대화에 실명, 주소 등 개인정보를 입력하지 않는다"
- **공통 규칙**: "민감한 개인정보를 입력하지 않는다", "AI 답변을 비판적으로 검토한다"

</StepCard>

> **만들 때 떠올려보자냥 (오늘 배운 세 가지!)**
> - 개인정보: 우리 앱 사용자가 위험한 정보를 넣지 않게 하려면?
> - ©️ 저작권: AI가 만든 결과물을 정직하게 밝히게 하려면?
> - 공정성: AI 답변을 무조건 믿지 않고 검토하게 하려면?

<Callout type="tip">
  **부담 갖지 말자냥!**

  가이드라인은 멋진 문장이 아니어도 된다냥. 짧고 분명한 **한 줄 규칙**이
  오히려 지키기 쉽다냥. 우리 모둠 앱에 딱 맞는 규칙 3개만 정해보자냥!
</Callout>

</MissionCard>

> **함께 만들어보자냥**
> - 모둠마다 겹치는 규칙(예: "개인정보 입력 금지")을 모아보자냥.
> - 그걸 모으면 **"우리 반 공통 AI 사용 규칙"**이라는 멋진 산출물이 완성된다냥!
