<MissionCard number={1} title="PDF 불러와서 벡터DB 채우기" time="8~10분">

이번엔 손으로 문장을 써넣는 대신, **진짜 PDF 파일**을 불러와서 도서관(벡터DB)을 채워볼 거다냥! 아래 템플릿에서 **빈칸(`____`)을 직접 채워보자냥.**

핵심은 **읽기 → 텍스트 추출 → 청킹 + 적재** 세 걸음이라냥. PDF에서 글자만 뽑으면, 그 다음은 지난 시간에 배운 것과 똑같다냥!

```python
import chromadb
from pypdf import PdfReader

client = chromadb.Client()
collection = client.get_or_create_collection(name="my_documents")

# 지난 시간(2차시)에 만든 청킹 함수 (없으면 이걸 그대로 쓰세요)
def split_into_chunks(text, chunk_size=200):
    chunks = []
    for i in range(0, len(text), chunk_size):
        chunks.append(text[i:i + chunk_size])
    return chunks

# 1) PDF 열기 — 본인의 PDF 파일 이름을 넣어보세요
reader = PdfReader(____)

# 2) 모든 페이지의 텍스트를 하나로 이어붙이기
text = ""
for page in reader.pages:
    text += ____   # 페이지에서 글자를 뽑는 함수를 호출해보세요

# 3) 청킹해서 벡터DB에 넣기
chunks = split_into_chunks(text, chunk_size=200)
collection.add(
    documents=chunks,
    ids=[f"chunk_{i}" for i in range(len(chunks))],
)
print(f"PDF에서 {len(chunks)}개의 청크를 도서관에 넣었다냥!")
```

> **확인해보자냥 (빈칸 힌트)**
> - `PdfReader(____)` 자리에는 PDF 파일 이름을 문자열로 넣으면 된다냥. 예: `PdfReader("교재.pdf")`
> - `text += ____` 자리에는 페이지에서 글자를 뽑는 `page.extract_text()`를 넣으면 된다냥.
> - 실행했을 때 "몇 개의 청크를 넣었다냥!"이 출력되면 성공이다냥!

<Callout type="note">
  글자가 하나도 안 뽑히냥? 그 PDF는 **사진(스캔) PDF**일 수 있다냥. 글자를 드래그해서 선택할 수 있는 PDF로 바꿔서 다시 해보자냥!

<MissionCard number={2} title="RAG 함수 만들기" time="8~10분">

이제 방금 PDF로 채운 도서관에서 검색(벡터DB)과 생성(제미나이)을 한 함수 안에서 합쳐볼 거다냥! 아래 템플릿에서 **빈칸(`____`)을 직접 채워보자냥.**

특히 prompt 안의 **역할** 빈칸이 중요하다냥. 지난 시간에 배운 "좋은 프롬프트 작성법"을 떠올리며 채워보자냥!

```python
import chromadb
from google import genai

genai_client = genai.Client(api_key="여기에 본인의 제미나이 API 키 입력")

# 미션 1에서 PDF로 채운 collection을 그대로 사용
client = chromadb.Client()
collection = client.get_or_create_collection(name="my_documents")

def rag_answer(question):
    # 1) 벡터DB에서 관련 청크 검색
    results = collection.query(query_texts=[question], n_results=2)
    retrieved_chunks = results["documents"][0]

    # 2) 검색된 내용을 하나의 문자열로 합치기
    context = "\n".join(retrieved_chunks)

    # 3) 역할/맥락/지시/형식을 담은 prompt를 직접 작성해보세요
    prompt = f"""
    너는 {____} 역할을 맡은 AI 도우미야.
    아래 [참고 자료]만을 근거로 질문에 답해줘.
    참고 자료에 없는 내용이면 "자료에서 찾을 수 없습니다"라고 답해줘.

    [참고 자료]
    {context}

    [질문]
    {question}
    """

    response = genai_client.models.generate_content(model="gemini-3.5-flash", contents=prompt)
    return response.text

# 질문을 직접 작성해 테스트해보세요
question = ____
print(rag_answer(question))
```

> **확인해보자냥 (빈칸 힌트)**
> - `너는 {____} 역할을` 자리에는 어떤 도우미인지 적으면 된다냥. 예: `"친절한 도서관 사서"`, `"우리 학교 안내 도우미"` 처럼!
> - `question = ____` 자리에는 문자열 질문을 넣으면 된다냥. 예: `question = "우리 학교 점심시간은 언제야?"`

<Callout type="tip">
  **재미있는 비교 실험!**

<MissionCard number={4} title="Streamlit으로 RAG 챗봇 완성하기" time="8~10분">

마지막이다냥! 미션 2에서 만든 `rag_answer` 함수를 Streamlit 화면에 연결해서, 입력창에 질문을 받아 답을 보여주는 **진짜 RAG 챗봇**을 완성해보자냥! 이제 이 챗봇은 **내가 넣은 PDF**를 근거로 답한다냥.

아래 템플릿에서 **빈칸(`____`)을 채워보자냥.**

```python
import streamlit as st

st.title("나만의 RAG 챗봇")
st.write("문서 자료를 바탕으로 질문에 답하는 AI입니다.")

user_question = st.text_input("질문을 입력하세요")

if st.button("질문하기"):
    answer = ____  # rag_answer 함수를 호출해보세요
    st.write("### 답변")
    st.write(answer)
    
    # (선택) 참고한 자료도 함께 보여주기
    with st.expander("참고한 자료 보기"):
        results = collection.query(query_texts=[user_question], n_results=2)
        for doc in results["documents"][0]:
            st.write("-", doc)
```

> **확인해보자냥 (빈칸 힌트)**
> - `answer = ____` 자리에는 미션 2에서 만든 함수를 호출하면 된다냥. 입력창의 값 `user_question`을 넘겨주면 된다냥!
> - 정답이 떠오르냥? `rag_answer(user_question)` 처럼 쓰면 된다냥.

**직접 해보기**

터미널에서 `streamlit run app.py`로 실행하고, 입력창에 여러 질문을 넣어보자냥. "참고한 자료 보기"를 펼쳐서, AI가 어떤 자료를 보고 답했는지도 구경해보자냥!
