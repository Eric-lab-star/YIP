<MissionCard number={1} title="RAG 함수 만들기 🔗" time="8~10분">

검색(벡터DB)과 생성(제미나이)을 한 함수 안에서 합쳐볼 거다냥! 아래 템플릿에서 **빈칸(`____`)을 직접 채워보자냥.**

특히 prompt 안의 **역할** 빈칸이 중요하다냥. 지난 시간에 배운 "좋은 프롬프트 작성법"을 떠올리며 채워보자냥!

```python
import chromadb
from google import genai

genai_client = genai.Client(api_key="여기에 본인의 제미나이 API 키 입력")

# 지난 시간 만든 Chroma DB와 collection을 그대로 사용
client = chromadb.Client()
collection = client.get_or_create_collection(name="my_documents")

# (지난 시간 데이터가 없다면 여기서 다시 add 해주세요)

def rag_answer(question):
    # 1) 벡터DB에서 관련 청크 검색
    results = collection.query(query_texts=[question], n_results=2)
    retrieved_chunks = results["documents"][0]

    # 2) 검색된 내용을 하나의 문자열로 합치기
    context = "\n".join(retrieved_chunks)

    # 3) 역할/맥락/지시/형식을 담은 prompt를 직접 작성해보세요
    prompt = f"""
    너는 {____} 역할을 맡은 AI 도우미야.
    아래 [참고 자료]만을 근거로 질문에 답해줘.
    참고 자료에 없는 내용이면 "자료에서 찾을 수 없습니다"라고 답해줘.

    [참고 자료]
    {context}

    [질문]
    {question}
    """

    response = genai_client.models.generate_content(model="gemini-3.5-flash", contents=prompt)
    return response.text

# 질문을 직접 작성해 테스트해보세요
question = ____
print(rag_answer(question))
```

> ✅ **확인해보자냥 (빈칸 힌트)**
> - `너는 {____} 역할을` 자리에는 어떤 도우미인지 적으면 된다냥. 예: `"친절한 도서관 사서"`, `"우리 학교 안내 도우미"` 처럼!
> - `question = ____` 자리에는 문자열 질문을 넣으면 된다냥. 예: `question = "우리 학교 점심시간은 언제야?"`

</MissionCard>

<MissionCard number={2} title="다양한 질문으로 테스트하기 🧪" time="8~10분">

이번 미션이 RAG의 진짜 가치를 느끼는 순간이라냥! 두 종류의 질문을 던져서 결과를 비교해보자냥.

<StepCard step="실험" title="이렇게 테스트해보자냥">

1. **문서에 있는 내용**을 질문하기 → 자료를 바탕으로 친절하게 답하는지 확인하기
2. **문서에 없는 내용**을 일부러 질문하기 → "자료에서 찾을 수 없습니다"라고 솔직하게 답하는지 확인하기

</StepCard>

> ✅ **확인해보자냥**
> - 문서에 있는 질문에는 자료 내용을 근거로 답하냥?
> - 문서에 없는 질문에는 아는 척하지 않고 "찾을 수 없습니다"라고 답하냥?

<Callout type="tip">
  🐾 **재미있는 비교 실험!**

  prompt에서 `참고 자료에 없는 내용이면 "자료에서 찾을 수 없습니다"라고 답해줘.` 이 한 줄을 **넣었을 때**와 **뺐을 때**의 결과를 비교해보자냥.

  이 줄을 빼면 AI가 자료에 없는 내용도 그럴듯하게 지어내기도 한다냥(이걸 환각이라고 부른다냥!). RAG에서 prompt 한 줄이 얼마나 중요한지 직접 느껴보자냥!
</Callout>

<MissionCard number={3} title="Streamlit으로 RAG 챗봇 완성하기 💬" time="8~10분">

마지막이다냥! 미션 1에서 만든 `rag_answer` 함수를 Streamlit 화면에 연결해서, 입력창에 질문을 받아 답을 보여주는 **진짜 RAG 챗봇**을 완성해보자냥!

아래 템플릿에서 **빈칸(`____`)을 채워보자냥.**

```python
import streamlit as st

st.title("나만의 RAG 챗봇")
st.write("문서 자료를 바탕으로 질문에 답하는 AI입니다.")

user_question = st.text_input("질문을 입력하세요")

if st.button("질문하기"):
    answer = ____  # rag_answer 함수를 호출해보세요
    st.write("### 답변")
    st.write(answer)
    
    # (선택) 참고한 자료도 함께 보여주기
    with st.expander("참고한 자료 보기"):
        results = collection.query(query_texts=[user_question], n_results=2)
        for doc in results["documents"][0]:
            st.write("-", doc)
```

> ✅ **확인해보자냥 (빈칸 힌트)**
> - `answer = ____` 자리에는 미션 1에서 만든 함수를 호출하면 된다냥. 입력창의 값 `user_question`을 넘겨주면 된다냥!
> - 정답이 떠오르냥? `rag_answer(user_question)` 처럼 쓰면 된다냥. 🐱

✍️ **직접 해보기**

터미널에서 `streamlit run app.py`로 실행하고, 입력창에 여러 질문을 넣어보자냥. "참고한 자료 보기"를 펼쳐서, AI가 어떤 자료를 보고 답했는지도 구경해보자냥!
