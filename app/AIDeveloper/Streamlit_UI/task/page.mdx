<MissionCard number={1} title="Streamlit 기본 환경 확인하기" time="5분">

먼저 Streamlit이 잘 깔렸는지, 화면이 잘 뜨는지 확인해보자냥. 새 파일 `app.py`를 만들고 아래 코드를 그대로 써보자냥.

```python
import streamlit as st

st.title("Hello, Streamlit! 🐱")
st.write("내 첫 번째 화면이 떴다냥!")
```

터미널에서 아래 명령어로 실행해보자냥.

```bash
streamlit run app.py
```

> ✅ **확인해보자냥**
> - 웹 브라우저가 자동으로 열리면서 화면이 나타났냥?
> - 제목과 글자가 화면에 잘 보이냥?

<Callout type="important">
  ⚠️ **화면이 안 뜨면?**

  가장 흔한 원인은 실행 명령어다냥! 터미널에서 `streamlit run 파일명.py` 를 정확히 입력했는지 다시 확인해보자냥. (그냥 `python app.py` 로 실행하면 화면이 안 뜬다냥!)
</Callout>

</MissionCard>

<MissionCard number={2} title="날씨 정보 화면 만들기 🌤️" time="10~12분">

지난 시간에 만든 **날씨 API 코드**를 Streamlit 화면에 연결해볼 거다냥. 도시 이름을 입력하고 버튼을 누르면 → 온도와 날씨가 화면에 나타나도록 만들어보자냥!

아래 템플릿에서 **빈칸(`____`, `___`)을 직접 채워보자냥.**

```python
import streamlit as st
import requests

st.title("오늘의 날씨 알리미")

city = st.text_input("도시 이름을 입력하세요 (예: Seoul)")

if st.button("날씨 확인하기"):
    weather_api_key = "여기에 본인의 API 키 입력"
    url = f"https://api.openweathermap.org/data/2.5/weather?q={city}&appid={____}&units=metric"

    response = requests.get(url)
    data = response.json()

    temperature = data["main"]["___"]  # 온도 값을 가져오는 키를 찾아보세요
    description = data["weather"][0]["description"]

    st.write(f"{city}의 현재 온도: {temperature}도")
    st.write(f"날씨 상태: {description}")
```

> ✅ **확인해보자냥 (빈칸 힌트)**
> - `{____}` 자리에는 어떤 변수가 들어가야 할까냥? (API 키를 담아둔 변수 이름이다냥!)
> - `data["main"]["___"]` 에서 온도 값을 가져오는 키는 무엇이냥? (영어로 "온도"라냥 🌡️)

✍️ **직접 해보기**
도시 이름을 `Seoul`, `Busan`, `Tokyo` 등으로 바꿔가며 입력하고 버튼을 눌러보자냥. 결과가 잘 바뀌냥?

<MissionCard number={3} title="간단한 AI 채팅 앱 만들기 💬" time="10~13분">

이번엔 제미나이 API를 화면에 연결해서, 질문하면 AI가 답해주는 **미니 챗봇**을 만들어볼 거다냥! (미션 2를 다 한 친구만 도전해도 좋다냥 😺)

```python
import streamlit as st
from google import genai

st.title("나만의 간단한 AI 챗봇")

client = genai.Client(api_key="여기에 본인의 제미나이 API 키 입력")

user_input = st.text_input("질문을 입력하세요")

if st.button("질문하기"):
    prompt = ____  # user_input을 활용한 prompt를 만들어보세요
    response = client.models.generate_content(model="gemini-3.5-flash", contents=prompt)
    st.write(response.text)
```

> ✅ **확인해보자냥 (빈칸 힌트)**
> - `prompt = ____` 자리에는 사용자가 입력한 `user_input`을 활용한 문장을 넣으면 된다냥.
> - 가장 간단하게는 `prompt = user_input` 으로 해도 되고, `prompt = f"{user_input} 쉽게 설명해줘"` 처럼 꾸며도 좋다냥!

<Callout type="tip">
  🐾 **이번 채팅 앱은 "한 번의 질문-답변"만 만들면 충분하다냥.** 대화 내용을 기억하는 챗봇은 다음 단계에서 배울 거니까, 지금은 질문 한 번 → 답변 한 번에만 집중하자냥!
</Callout>

</MissionCard>
