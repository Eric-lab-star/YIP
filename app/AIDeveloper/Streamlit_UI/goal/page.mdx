### 🍽️ 주방과 테이블로 이해하는 Streamlit

지난 시간까지 우리는 `.ipynb`라는 **주방**에서 요리(코드 실행 결과)를 만들었다냥. 그런데 그 요리를 손님(사용자)에게 보여줄 **예쁜 그릇과 테이블**이 없었다냥.

Streamlit은 바로 그 **테이블 세팅 담당**이다냥! 주방에서 만든 데이터를, 손님이 보기 좋은 화면(글자, 입력창, 버튼)에 자동으로 차려주는 역할이라냥.

<Callout type="analogy">
  **비유로 이해하기**

  | 식당 | 우리 코드 |
  |---|---|
  | 주방에서 만든 요리 | `.ipynb`에서 실행한 코드 결과 |
  | 예쁜 그릇 · 테이블 세팅 | Streamlit 화면 (글자, 입력창, 버튼) |
  | 가게에 들어온 손님 | 우리 앱을 쓰는 사용자 |

  코드 몇 줄만 추가하면, 누구나 클릭하고 입력할 수 있는 진짜 "앱"이 완성된다냥!
</Callout>

## 2단계 · Streamlit 기본 사용법 (5~7분)

<NyangSpeech>
  자, 이제 Streamlit을 쓰는 법을 알려줄 거다냥. 딱 **설치 → 실행 → 함수 3~4개**만
  알면 화면을 만들 수 있다냥. 생각보다 진짜 간단하다냥! 😺
</NyangSpeech>

### 📦 설치하고 실행하기

```bash
# 1) 설치 (처음 한 번만)
pip install streamlit

# 2) 실행 — app.py 파일을 웹 화면으로 띄우기
streamlit run app.py
```

`streamlit run app.py`를 실행하면 자동으로 웹 브라우저가 열리면서 우리가 만든 화면이 나타난다냥!

### 🧰 꼭 알아야 할 핵심 함수

<StepCard step="함수" title="이 4개만 알면 화면을 만들 수 있다냥!">

| 함수 | 하는 일 |
|---|---|
| `st.title("제목")` | 큰 **제목**을 화면에 보여주기 |
| `st.write("내용")` | 글자·숫자·표 등 **거의 모든 것**을 출력하기 (만능 출력 함수!) |
| `st.text_input("안내문")` | 사용자가 글자를 입력하는 **입력창** 만들기 |
| `st.button("버튼이름")` | **버튼**을 만들고, 클릭했는지 확인하기 |

</StepCard>

### 🐣 기본 코드 예시

```python
import streamlit as st

st.title("나의 첫 Streamlit 앱")
st.write("안녕하세요! 이것은 Streamlit으로 만든 화면입니다.")

name = st.text_input("이름을 입력하세요")
if st.button("인사하기"):
    st.write(f"{name}님, 반갑습니다!")
```

이 코드를 실행하면 → 제목, 안내 글, 이름 입력창, 그리고 "인사하기" 버튼이 있는 화면이 뿅! 하고 나타난다냥. 버튼을 누르면 입력한 이름으로 인사까지 해준다냥! 🎉

<Callout type="tip">
  💡 **꼭 기억할 두 가지**

  - 코드를 수정하고 **저장**하면, 화면 오른쪽 위의 **`Rerun`** 버튼으로 새 화면을 다시 그릴 수 있다냥.
  - `st.write()`는 글자, 숫자, 표, 딕셔너리까지 거의 다 출력할 수 있는 **만능 출력 함수**라냥. 헷갈리면 일단 `st.write()`부터 써보자냥!
</Callout>
