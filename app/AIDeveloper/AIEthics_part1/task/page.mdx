<MissionCard number={1} title="환각 유도 실험 — AI가 지어내는 모습 관찰하기 🔍" time="8~10분">

먼저 AI가 **존재하지 않는 것**에 대해 질문받으면 어떻게 반응하는지 직접 관찰해보자냥. 아래 코드를 그대로 실행해보자냥.

```python
from google import genai

client = genai.Client(api_key="여기에 본인의 제미나이 API 키 입력")

# 1) 환각을 유도하는 질문 (존재하지 않을 법한 것)
hallucination_prompt = "2024년에 출판된 책 '용감한 코딩 고양이의 모험'의 줄거리를 알려줘."
print("=== 안전장치 없는 답변 ===")
print(client.models.generate_content(model="gemini-3.5-flash", contents=hallucination_prompt).text)
```

> ✅ **관찰해보자냥**
> - AI가 **"그런 책은 모른다"**고 답했냥, 아니면 **줄거리를 그럴듯하게 지어냈냥?**
> - 지어냈다면, 그 답이 얼마나 **자연스럽고 자신만만**한지 살펴보자냥. 바로 그게 환각이라냥! 😼

<Callout type="tip">
  🐾 **좋은 환각 실험 주제 고르는 법**

  "존재하지 않는 책 / 영화 / 인물"처럼 **내가 직접 진짜인지 확인할 수 있는** 주제를 고르자냥. 그래야 AI가 지어냈는지 아닌지 바로 알 수 있다냥!
</Callout>

</MissionCard>

<MissionCard number={2} title="프롬프트로 환각 줄이기 — 안전장치 추가하기 🛡️" time="8~10분">

이번엔 같은 질문에 **"모르면 모른다고 답하라"**는 지시를 넣어서, 답변이 어떻게 달라지는지 비교해보자냥!

아래 템플릿에서 **빈칸(`____`)을 직접 채워보자냥.**

```python
# 2) 환각을 줄이는 지시를 직접 추가해보세요
safe_prompt = f"""
{____}
질문: 2024년에 출판된 책 '용감한 코딩 고양이의 모험'의 줄거리를 알려줘.
"""
# 힌트: "확실하지 않거나 사실인지 모르는 정보는 지어내지 말고, '확인할 수 없다'고 답해줘."

print("\n=== 안전장치 있는 답변 ===")
print(client.models.generate_content(model="gemini-3.5-flash", contents=safe_prompt).text)
```

> ✅ **확인해보자냥 (빈칸 힌트)**
> - `{____}` 자리에는 AI에게 주는 **안전장치 지시문**이 들어간다냥.
> - 힌트의 문장을 그대로 써봐도 좋다냥: *"확실하지 않거나 사실인지 모르는 정보는 지어내지 말고, '확인할 수 없다'고 답해줘."*

✍️ **직접 비교해보기**

미션 1의 **안전장치 없는 답변**과, 미션 2의 **안전장치 있는 답변**을 **나란히** 놓고 비교해보자냥. 지시 한 줄을 넣었을 뿐인데 답이 달라지냥? 그 차이가 바로 안전장치의 힘이라냥! 💪

</MissionCard>

<MissionCard number={3} title="모둠 토론 — 우리 앱에 어떤 안전장치를 넣을까냥? 💬" time="8~10분">

이번엔 코드를 잠시 내려놓고, 친구들과 함께 **토론**해보는 미션이다냥! 아래 세 가지 주제를 모둠끼리 이야기 나눠보자냥.

```text
1. 우리가 만든 수학 풀이 봇이 틀린 답을 내놓으면 어떤 문제가 생길까?
2. 사용자에게 "이 답은 AI가 만든 것이라 틀릴 수 있다"는 점을 어떻게 알려주면 좋을까?
3. 어떤 종류의 질문에는 AI 답변을 특히 더 조심해야 할까? (의료, 법률, 안전 등)
```

<Callout type="note">
  📝 **토론은 정답을 찾는 게 아니라냥**

  토론의 목적은 하나의 정답을 찾는 게 아니라, **다양한 관점을 서로 나누는 것**이라냥. 친구의 생각이 나와 달라도 좋다냥. 서로의 아이디어를 듣고 더 좋은 안전장치를 떠올려보자냥!
</Callout>

</MissionCard>

## ★ 핵심 비유: "안전벨트를 채운 자동차" 🚗

<NyangSpeech>
  실습을 하면서 느꼈냥? AI는 정말 편리하지만, 가끔 사고(환각)를 낸다냥.
  그렇다고 안 쓸 수는 없으니, 우리만의 **안전벨트**를 채워주는 거라냥! 🐾
</NyangSpeech>

<StepCard step="비유" title="자동차(AI)에 안전벨트(검토 습관)와 신호등(안전장치)을 달자냥 🚦">

자동차(**AI**)는 아주 편리하지만, 사고(**환각**)가 날 수도 있다냥.

그렇다고 차를 안 탈 수는 없지냥? 대신 우리는 **안전벨트(검토 습관)**와 **신호등(안전장치)**을 만든다냥.

- 🪢 **모르면 모른다고 답하기** (프롬프트 지시)
- 📚 **참고 자료를 근거로 답하기** (RAG)
- 🏷️ **AI 답변임을 표시하기** (사용자에게 알려주기)

이런 장치들이 바로 AI를 **안전하게 타기 위한 안전벨트**인 거라냥!

</StepCard>
