## 미션 1. 첫 코드 셀 실행해보기 (5분)

## 미션 3. 셀 사이에서 변수가 살아있는지 확인하기 (5분)

`.ipynb`의 진짜 매력은, **한 셀에서 만든 변수를 다른 셀에서도 계속 쓸 수 있다**는 거다냥! 직접 확인해보자냥.

**셀 1** (먼저 실행)
```python
my_name = "코딩냥"
```

**셀 2** (그 다음에 실행)
```python
print(my_name, "안녕!")
```

순서대로 **셀 1 → 셀 2** 실행하면 `코딩냥 안녕!` 이라는 결과가 나타날 거다냥.

> **확인해보자냥 (실험 시간!)**
> 1. 셀 2를 먼저 실행하면 어떻게 되냥? (에러가 나는지 확인해보자냥)
> 2. 셀 1을 실행한 뒤, `my_name` 값을 다른 이름으로 바꿔서 다시 실행해보자냥. 그 다음 셀 2를 다시 실행하면 결과가 바뀌냥?
> 3. 이걸 통해 알 수 있는 건, `.ipynb`에서는 **셀의 위치보다 "실행한 순서"가 더 중요하다**는 사실이다냥!

**생각해보기**
셀을 순서 없이 막 실행하면 어떤 일이 생길 수 있을까냥? 한 줄로 적어보자냥.

## 미션 4. 간단한 AI 예제 코드 실행해보기 (5분)

이번엔 진짜 AI스러운 코드를 실행해볼 거다냥! 아래 예제는 입력한 문장에 따라 간단하게 반응을 보여주는 코드다냥.

```python
def simple_ai_response(text):
    if "안녕" in text:
        return "AI: 안녕하세요! 반가워요 😊"
    elif "고양이" in text:
        return "AI: 저도 고양이를 좋아해요! 🐱"
    elif "AI" in text or "인공지능" in text:
        return "AI: 저는 인공지능에 대해 이야기하는 걸 좋아해요!"
    else:
        return "AI: 음... 잘 이해하지 못했어요. 다른 말을 해볼래요?"

# 아래 문장을 자유롭게 바꿔보자냥!
print(simple_ai_response("안녕! 너는 고양이를 좋아하니?"))
```

새 셀에 위 코드를 그대로 입력하고 실행해보자냥.

> **확인해보자냥**
> - `simple_ai_response("...")` 안의 문장을 다른 문장으로 바꿔서 실행해보자냥. 결과가 어떻게 달라지냥?
> - 이 코드는 사람이 미리 정해둔 규칙(`if`, `elif`)에 따라 답을 골라주는 거다냥. 실제 AI는 이것보다 훨씬 복잡한 방식으로 동작하지만, **"입력을 받아서 → 처리하고 → 결과를 출력한다"**는 큰 흐름은 비슷하다냥!

**나만의 규칙 추가하기**
`elif` 줄을 하나 추가해서, 특정 단어가 들어가면 내가 원하는 답이 나오도록 만들어보자냥. (예: "날씨"가 들어가면 "AI: 오늘 날씨 정보를 알려드릴게요!" 같은 답)
