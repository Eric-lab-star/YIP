### 💬 카톡 같은 화면을 만드는 두 함수

<StepCard step="함수" title="이 두 개면 채팅 화면이 뚝딱이다냥!">

| 함수 | 하는 일 |
|---|---|
| `st.chat_input("안내문")` | 화면 아래쪽에 **채팅 입력창**을 만들어주기 |
| `st.chat_message("user")` | **말풍선**을 만들어 그 안에 메시지를 보여주기 (`"user"` / `"assistant"`) |

</StepCard>

### 🧠 화면이 새로고침돼도 대화가 안 사라지게 — 세션 상태

  **싹 사라져버린다냥**. 그걸 막아주는 게 바로 `st.session_state`라는 **기억 상자**다냥!

### 🪧 핵심 비유 — "대화 내용을 잊지 않는 칠판"

Streamlit 화면은 사실 새로운 입력이 있을 때마다 칠판을 **싹 지우고** 처음부터 다시 그린다냥. 그냥 두면 이전 대화가 다 사라져버리겠지냥?

그래서 우리는 `session_state`라는 **"지워지지 않는 칠판 한 켠"**에 대화 기록을 적어둔다냥. 화면을 다시 그릴 때마다, 이 칠판에 적힌 대화를 처음부터 쭉 다시 보여주는 거다냥. 덕분에 사용자는 대화가 끊기지 않고 **이어지는 것처럼** 느낄 수 있다냥! 🐱

<Callout type="important">
  ⚠️ **세션 상태의 핵심 개념**

  `session_state`는 **"화면이 새로 그려져도 유지되는 데이터 저장소"**다냥. 이 부분이 없으면 대화가 **매번 사라진다냥**. 나중에 실습에서 일부러 빼보면 차이를 확실히 느낄 수 있다냥!
</Callout>

### 🐣 채팅 화면 기본 코드 살펴보기

```python
import streamlit as st

# 세션 상태에 대화 기록 저장소 만들기 (없을 때만 빈 리스트로)
if "messages" not in st.session_state:
    st.session_state.messages = []

# 지금까지의 대화를 화면에 다시 그리기
for msg in st.session_state.messages:
    with st.chat_message(msg["role"]):
        st.write(msg["content"])

# 화면 아래쪽 채팅 입력창
user_input = st.chat_input("영어로 말해보세요")
```

`if "messages" not in st.session_state:` 부분이 바로 **"칠판이 처음이면 빈 칸으로 준비해두자"**는 뜻이다냥. 그리고 `for` 반복문으로 칠판에 적힌 대화를 **처음부터 쭉** 말풍선으로 다시 그려주는 거다냥! 🐾

### 🪧 핵심 비유 — "대화도 하고 첨삭도 해주는 선생님"

좋은 영어 선생님은 대화만 나누는 게 아니라, 학생이 한 말을 **자연스럽게 고쳐주기도** 한다냥. 오늘 추가하는 교정 기능이 바로 그 역할이다냥.

학생이 영어로 말하면, 대화는 대화대로 이어가면서 *"이 표현은 이렇게 하면 더 자연스러워요"*라고 살짝 **첨삭**까지 해주는 친절한 선생님이 완성되는 거다냥! 🐟
