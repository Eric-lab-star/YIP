<MissionCard number={1} title="BotFather로 봇 만들고 토큰 받기 🔑" time="8~10분">

가장 먼저, 내 봇을 만들고 **열쇠(토큰)**를 발급받아야 한다냥. 텔레그램 앱에서 아래 순서대로 따라 해보자냥!

<StepCard step="순서" title="봇 생성 5단계">

1. 텔레그램 앱에서 검색창에 **`BotFather`**를 검색한다냥. (파란 체크 표시가 있는 공식 계정이라냥!)
2. BotFather와 대화를 시작하고, **`/newbot`**이라고 메시지를 보낸다냥.
3. 봇의 **이름(name)**을 정해서 보낸다냥. (화면에 보일 이름이라냥)
4. 봇의 **사용자명(username)**을 정한다냥. 반드시 **`bot`으로 끝나야** 한다냥! (예: `my_math_bot`)
5. BotFather가 축하 메시지와 함께 **긴 토큰**을 보내준다냥. 이걸 잘 복사해두자냥!

</StepCard>

> ✅ **확인해보자냥**
> - BotFather가 보내준 토큰을 받았냥? (`123456789:ABC...` 같은 긴 문자열이라냥)
> - 토큰을 메모장 등에 안전하게 복사해두었냥?

<Callout type="important">
  ⚠️ **토큰은 비밀번호다냥!**

  이 토큰은 내 봇을 여는 **열쇠**라서, 친구에게 보여주거나 인터넷에 올리면 안 된다냥. 만약 실수로 노출됐다면, BotFather에서 `/revoke`로 토큰을 새로 발급받을 수 있다냥!
</Callout>

</MissionCard>

<MissionCard number={3} title="메아리 봇 만들고 실행하기 🦜" time="12~15분">

이제 핵심이다냥! 사용자가 보낸 메시지를 **그대로 따라 답장**하는 메아리 봇을 만들어볼 거다냥.

아래 템플릿에서 **빈칸(`____`)을 직접 채워보자냥.** 그리고 `TELEGRAM_TOKEN`에는 미션 1에서 받은 **내 토큰**을 붙여넣자냥!

```python
from telegram import Update
from telegram.ext import Application, MessageHandler, CommandHandler, filters, ContextTypes

TELEGRAM_TOKEN = "여기에 BotFather에서 받은 토큰 입력"

# /start 명령어에 응답하는 함수
async def start(update: Update, context: ContextTypes.DEFAULT_TYPE):
    await update.message.reply_text("안녕하세요! 메시지를 보내면 따라 말해드릴게요.")

# 일반 메시지에 응답하는 함수
async def echo(update: Update, context: ContextTypes.DEFAULT_TYPE):
    user_text = update.message.text
    # 받은 메시지를 그대로 답장하도록 작성해보세요
    await update.message.reply_text(____)

# 봇 실행
app = Application.builder().token(TELEGRAM_TOKEN).build()
app.add_handler(CommandHandler("start", start))
app.add_handler(MessageHandler(filters.TEXT & ~filters.COMMAND, echo))

print("봇이 실행 중입니다...")
app.run_polling()
```

> ✅ **확인해보자냥 (빈칸 힌트)**
> - `reply_text(____)`의 빈칸에는 무엇이 들어가야 할까냥? (사용자가 보낸 메시지를 담아둔 **변수 이름**이다냥! 윗줄을 잘 보자냥 👀)

✍️ **직접 해보기**
코드를 저장하고 실행한 다음, **내 텔레그램에서 직접 내 봇을 검색**해보자냥! (미션 1에서 정한 username으로 찾으면 된다냥) 그리고 아무 말이나 보내보자냥. 봇이 그대로 따라 말하냥? 🎉

</MissionCard>

---

<Callout type="tip">
  🐾 **봇을 멈추고 싶을 땐?**

  `app.run_polling()`은 봇을 **계속 켜둔 상태**로 둔다냥. 그래서 코드가 멈추지 않고 계속 메시지를 기다린다냥. 봇을 멈추려면 **셀 중단** 또는 터미널에서 **`Ctrl+C`**를 눌러야 한다냥!
</Callout>
