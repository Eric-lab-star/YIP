### 📒 대화 내용을 적어둔 수첩을 들고 다니는 사람

AI는 사실 대화를 나눌 때마다 기억이 사라지는 사람과 같다냥. 방금 한 말도 금방 잊어버린다냥.

그래서 우리는 AI에게 매번 **"지금까지 나눈 대화를 적어둔 수첩"**을 함께 건네준다냥. AI는 그 수첩을 쭉 읽어본 뒤, 흐름에 맞는 다음 말을 이어가는 거다냥.

이 수첩(**대화 히스토리**)을 잘 관리하는 게 바로 자연스러운 대화 앱의 핵심이라냥!

<Callout type="analogy">
  **비유로 이해하기 — 기억이 사라지는 사람과 수첩**

  | 상황 | 우리 코드 |
  |---|---|
  | 매번 기억이 사라지는 사람 | 이전 대화를 기억 못 하는 AI |
  | 지금까지의 대화를 적은 수첩 | 대화 히스토리(history) |
  | 수첩을 읽고 다음 말을 이어가기 | 과거 대화를 함께 보내 맥락 유지하기 |

  "AI가 대화를 기억한다"는 건 사실 **"매번 과거 대화를 다시 보내주는 것"**이라냥!
</Callout>

## 2단계 · 페르소나로 "나만의 영어 선생님" 임명하기 (5분)

<NyangSpeech>
  수첩만 있으면 끝이 아니다냥! AI에게 **어떤 역할**을 맡길지도 정해줘야 한다냥.
  이걸 **페르소나(persona)**라고 부른다냥. "너는 친절한 영어 회화 선생님이야"라고
  미리 알려주면, AI가 그 역할에 맞춰 대화해준다냥! 😺
</NyangSpeech>

페르소나는 AI에게 미리 건네는 **역할 설명서**라냥. 역할, 난이도, 말투를 정해주면 학습에 딱 맞는 대화 상대가 된다냥.

<StepCard step="페르소나" title="좋은 영어 선생님 페르소나에 담을 것">

- **역할**: "너는 친절한 영어 회화 선생님이야."
- **난이도**: "초급 학습자에게 쉬운 단어로 천천히 말해줘."
- **고쳐주기**: "학생이 실수하면 자연스럽게 고쳐줘."
- **한국어 설명**: "답변 끝에 한국어로 짧은 설명을 덧붙여줘."

</StepCard>

이런 조건을 넣을수록 **학습용으로 더 적합한 대화**가 된다냥. (지난 프롬프트 수업에서 배운 내용과 연결되는 부분이라냥!)

### 🐣 기본 코드 예시 — 페르소나 설정하기

```python
import google.generativeai as genai

genai.configure(api_key="여기에 본인의 제미나이 API 키 입력")

system_instruction = "너는 친절한 영어 회화 선생님이야. 초급 학습자에게 쉬운 단어로 천천히 영어로 말해주고, 학생이 실수하면 자연스럽게 고쳐줘. 답변 끝에 한국어로 짧은 설명을 덧붙여줘."

model = genai.GenerativeModel(
    "gemini-1.5-flash",
    system_instruction=system_instruction
)

response = model.generate_content("Hello! Can we talk about my weekend?")
print(response.text)
```

`system_instruction`에 페르소나를 적어두면, AI는 매번 그 역할을 기억하고 대화한다냥!

<Callout type="tip">
  💡 페르소나에 **"난이도(초급/중급)"**, **"실수 고쳐주기"**, **"한국어 설명 추가"** 같은
  조건을 넣을수록 영어 공부에 더 도움이 되는 선생님이 된다냥. 나만의 선생님을 마음껏 디자인해보자냥!
</Callout>
