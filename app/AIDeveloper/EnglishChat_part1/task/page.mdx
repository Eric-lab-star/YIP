<MissionCard number={2} title="페르소나 설정하고 한 번 대화하기 🧑‍🏫" time="10~12분">

먼저 AI에게 **영어 선생님 역할(페르소나)**을 부여하고, 한 번 대화를 나눠볼 거다냥. 아래 템플릿에서 **빈칸(`____`)을 직접 채워보자냥.**

```python
from google import genai
from google.genai import types

client = genai.Client(api_key="여기에 본인의 제미나이 API 키 입력")

# 페르소나(시스템 지시)를 직접 작성해보세요
system_instruction = ____
# 예: "너는 친절한 영어 회화 선생님이야. 초급 학습자에게 쉬운 단어로 천천히 영어로 말해주고,
#      학생이 실수하면 자연스럽게 고쳐줘. 답변 끝에 한국어로 짧은 설명을 덧붙여줘."

response = client.models.generate_content(
    model="gemini-3.5-flash",
    contents="Hello! Can we talk about my weekend?",
    config=types.GenerateContentConfig(system_instruction=system_instruction)
)
print(response.text)
```

> ✅ **확인해보자냥 (빈칸 힌트)**
> - `system_instruction = ____` 자리에는 **페르소나를 적은 문자열**을 넣으면 된다냥. (예시 주석을 참고해서 따옴표 `"..."` 안에 적어보자냥!)
> - 역할, 난이도, 고쳐주기, 한국어 설명 같은 조건을 넣을수록 좋은 선생님이 된다냥.

✍️ **직접 해보기**
나만의 페르소나로 바꿔보자냥. "중급 학습자용", "여행 영어 전문", "친구처럼 편한 말투" 등 원하는 선생님을 디자인해보자냥!

</MissionCard>

<MissionCard number={3} title="대화 기록 유지하며 여러 번 대화하기 📒" time="10~13분">

이번엔 **대화 히스토리(수첩)**를 유지하면서 여러 번 대화를 주고받아볼 거다냥! 제미나이의 `chat` 기능을 쓰면 히스토리가 자동으로 관리된다냥.

아래 템플릿에서 **빈칸(`____`)을 직접 채워보자냥.**

```python
# 제미나이의 chat 기능을 사용하면 대화 히스토리가 자동으로 관리됩니다
chat = client.chats.create(
    model="gemini-3.5-flash",
    config=types.GenerateContentConfig(system_instruction=system_instruction)
)

# 여러 번 대화 주고받기
while True:
    user_input = input("You: ")
    if user_input.lower() == "quit":
        break

    # 사용자 입력을 보내고 응답을 받도록 작성해보세요
    response = ____
    print("Teacher:", response.text)
```

> ✅ **확인해보자냥 (빈칸 힌트)**
> - `response = ____` 자리에는 **`chat` 객체로 메시지를 보내는 코드**가 들어간다냥. (`chat.send_message(...)` 형태를 떠올려보자냥! 안에는 `user_input`을 넣으면 된다냥.)
> - `quit`을 입력하면 대화가 끝난다냥. 그 전까지는 계속 주고받을 수 있다냥!

✍️ **직접 해보기**
영어로 자기소개를 한 뒤, 취미나 주말 이야기로 대화를 이어가보자냥. AI가 **앞에서 한 내 말을 기억하고** 이어주는지 확인해보자냥!

<Callout type="tip">
  🐾 **`start_chat`은 어떻게 기억하는 거냥?**

  `start_chat`을 쓰면 히스토리가 자동으로 관리되어 정말 편하다냥. 하지만 속을
  들여다보면, 내부적으로는 우리가 배운 그 원리 그대로 **"과거 대화를 매번 함께
  보내주는 것"**이라냥. 수첩을 자동으로 들고 다녀주는 셈이라냥! 📒
</Callout>
