먼저 `google-generativeai` 패키지가 잘 깔렸는지, 제미나이 API 키가 준비됐는지 확인해보자냥. 새 파일을 만들고 아래 코드가 에러 없이 실행되는지 점검해보자냥.

```python
import google.generativeai as genai

genai.configure(api_key="여기에 본인의 제미나이 API 키 입력")

```

> ✅ **확인해보자냥**
> - `import` 에서 에러가 나면 → 터미널에서 `pip install google-generativeai` 를 다시 해보자냥.
> - API 키 자리에 본인의 키를 정확히 붙여넣었냥?

<MissionCard number={2} title="블로그 구조 생성하기 🏗️" time="10~12분">

이제 AI에게 **글의 설계도(제목 + 소제목 3개)**를 만들어달라고 해보자냥! 아래 템플릿에서 **빈칸(`____`)을 직접 채워보자냥.**

```python
import google.generativeai as genai

genai.configure(api_key="여기에 본인의 제미나이 API 키 입력")
model = genai.GenerativeModel("gemini-1.5-flash")

topic = "초보자를 위한 홈카페 만들기"

# 역할/형식을 지정한 프롬프트를 직접 완성해보세요
structure_prompt = f"""
너는 {____} 말투의 블로거야.
'{topic}'이라는 주제로 네이버 블로그 글을 쓰려고 해.
다음 형식으로 글의 구조만 먼저 만들어줘:

제목: (눈길을 끄는 제목 1개)
소제목1: 
소제목2: 
소제목3: 
"""

structure = model.generate_content(structure_prompt).text
print(structure)
```

> ✅ **확인해보자냥 (빈칸 힌트)**
> - `{____}` 자리에는 **말투**를 넣으면 된다냥. `친근한`, `전문적인`, `유머러스한` 중에서 골라보자냥!
> - 실행하면 제목 1개와 소제목 3개가 깔끔하게 나오냥?

✍️ **직접 해보기**
`topic` 을 본인이 쓰고 싶은 주제(예: `제주도 2박 3일 여행`, `자취생 5분 요리`)로 바꿔보자냥. 말투도 바꿔가며 결과가 어떻게 달라지는지 살펴보자냥! 🐟

</MissionCard>

<MissionCard number={3} title="본문 채우기 ✍️" time="10~13분">

설계도가 완성됐으니, 이제 **각 소제목마다 본문**을 채워서 한 편의 글로 합쳐볼 거다냥! 위에서 나온 소제목들을 복사해서 아래 리스트에 넣고, **빈칸(`____`)을 채워보자냥.**

```python
# 위에서 만든 소제목들을 리스트로 정리 (직접 입력하거나 파싱)
subtitles = [
    "소제목1 내용을 여기에",
    "소제목2 내용을 여기에",
    "소제목3 내용을 여기에",
]

full_post = f"[주제: {topic}]\n\n"

for subtitle in subtitles:
    # 각 소제목에 맞는 본문 생성 프롬프트를 직접 작성해보세요
    body_prompt = f"""
    '{topic}' 블로그 글의 '{subtitle}' 부분 본문을 써줘.
    {____} 분량으로, 친근한 말투로 작성해줘.
    """
    body = model.generate_content(body_prompt).text
    full_post += f"## {subtitle}\n{body}\n\n"

print(full_post)
```

> ✅ **확인해보자냥 (빈칸 힌트)**
> - `{____}` 자리에는 **본문 분량**을 넣으면 된다냥. `3문장`, `200자` 처럼 구체적으로 적어보자냥!
> - 반복문이 소제목 3개를 하나씩 AI에게 건네주며 본문을 받아온다냥. 전체 글이 잘 합쳐졌냥?

### 📚 핵심 비유 · 목차를 먼저 짜고 챕터를 쓰는 작가

<Callout type="analogy">
  좋은 작가는 책을 쓸 때 먼저 **목차(소제목)**를 정하고, 그다음 각 **챕터(본문)**를
  하나씩 써내려간다냥. 📖

  우리 앱도 똑같이 일한다냥! 먼저 AI에게 목차를 만들게 하고(미션 2), 그 목차를 하나씩
  다시 AI에게 건네주며 "이 챕터 본문 써줘"라고 부탁하는 거다냥(미션 3). 이렇게 일을
  나누면 각 부분에 더 집중해서 좋은 글을 쓸 수 있다냥! 🐱
</Callout>
