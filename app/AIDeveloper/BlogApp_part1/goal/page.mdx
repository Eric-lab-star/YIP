### 프롬프트 네 가지 재료를 블로그에 적용하기

<StepCard step="프롬프트" title="이렇게 네 가지를 채워주면 된다냥!">

| 재료 | 블로그 글에 적용한 예시 |

여기서 가장 중요한 비법은 **"한 번에 다 시키지 않는 것"**이다냥! 긴 글 전체를 한 방에 요청하기보다, **먼저 구조(뼈대)를 잡고 → 그다음 살(본문)을 붙이는** 방식으로 나눠서 시키는 거다냥.

## 3단계 · 코드로 살짝 맛보기 (기본 구조)

<NyangSpeech>
  실제 코드는 실습 페이지에서 직접 빈칸을 채워가며 만들 거다냥. 여기서는 "아, 이런
  모양이구나~" 하고 눈에 익히기만 하면 된다냥!
</NyangSpeech>

먼저 AI에게 **글의 구조(설계도)**만 만들어달라고 부탁하는 코드다냥. 제목과 소제목 3개를 받아오는 모습이라냥.

```python
from google import genai

client = genai.Client(api_key="여기에 본인의 제미나이 API 키 입력")

topic = "초보자를 위한 홈카페 만들기"

structure_prompt = f"""
너는 친근한 말투의 블로거야.
'{topic}'이라는 주제로 네이버 블로그 글을 쓰려고 해.
다음 형식으로 글의 구조만 먼저 만들어줘:

제목: (눈길을 끄는 제목 1개)
소제목1:
소제목2:
소제목3:
"""

structure = client.models.generate_content(model="gemini-3.5-flash", contents=structure_prompt).text
print(structure)
```

이렇게 **형식을 콕 집어서** 알려주면, AI는 우리가 원하는 모양대로 깔끔하게 구조를 만들어준다냥. 이게 바로 "설계도 그리기" 단계라냥!

<Callout type="tip">
  **말투를 바꾸면 글의 느낌이 확 달라진다냥!**

  `너는 친근한 말투의 블로거야` 대신 `전문적인 말투`, `유머러스한 말투`로 바꿔보자냥.
  같은 주제라도 완전히 다른 글이 나오는 걸 직접 느낄 수 있다냥!
</Callout>
