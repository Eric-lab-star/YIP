# 반복문 Loop

한번 농부가 되었다고 상상을 해봐요. 밭에 당근을 심으려고 모든 준비를 끝낸 상태에요. 당근 심기 작업 순서는 3단계로 이루어져 있어요.

1. 땅에 구멍을 판다.
2. 당근 씨앗을 심는다.
3. 물을 뿌린다.

정말 간단하네요. 하지만 안 좋은 소식은 심어야 할 당근이 256개라는 거예요. 사람이라면 오늘 못 끝내지만, 컴퓨터라면 가능하지 않을까요? 컴퓨터는 이런 반복적인 작업을 매우 잘해요. 여러분들이 자주 하는 게임이나 동영상은 화면을 1초에 60번 이상 새로 그리고 있답니다.

## 기본구조

같은 작업을 여러 번 반복해서 하는 코드를 반복문이라고 해요. 파이썬에는 다양한 반복문이 있는데 그 중에서 2가지만 배울 거예요. 하나는 `while`이고 다른 하나는 `for..in`이에요.

<span id="while" />

### while

```python
# while 반복문
count = 3
while count > 0:
    print(count)
    count = count - 1
if count == 0:
    print(" 🚀 로켓 발사!~~~~~")
```

먼저 확인할 것은 `while`이에요. `while` 코드 다음에는 조건을 입력해요. 그리고 입력한 조건이 참이면 계속 `while`을 실행하고, 거짓이면 `while` 코드 실행을 종료해요.

이 코드를 위에서부터 살펴보면 `count`의 값이 처음에 3이에요. 그 다음에 `while`문을 컴퓨터가 읽게 됩니다. `while` 뒤에 오는 조건이 참이면, `while` 내부에 있는 실행문을 실행합니다. 현재 count의 값이 3이므로 `count > 0`이 참이 됩니다. 조건이 참이므로 `while` 내부의 코드가 조건이 거짓일 때까지 실행됩니다.

### for ... in

```python
# for 루프
for i in range(0, 4):
    print(i)
```

다음에 배워볼 루프는 `for ..in` 루프에요. `range()`는 0에서부터 그 이전까지의 숫자의 범위를 의미해요. 그래서 `for i in range(0, 5)`를 한국어로 번역하면, "0에서 5 사이의 변수 i에 대해서"라는 의미에요. 만약 변수 i가 0에서 5 사이의 숫자가 아니라면 루프 내부의 코드를 실행하지 않아요.

`for ... in` 또한 다양한 곳에서 많이 사용돼요. 다양한 곳에서 사용될 수 있기 때문에 잘 알아두세요.

### 연습하기

```python
k = 1
while k < 6:
    print(f"{' . ' * k:░>19}", end="")
    print(f"{' . ' * k:░<19}", )
    k += 1
```

이 코드를 실행했을 때 피라미드 모양이 나와요. 피라미드를 뒤집어서 모래시계 모양이 나오게 코드를 작성해 보세요.

## 강제종료 — break

이제 앞에서 배운 `while` 또는 `for .. in` 루프를 사용해서 당근을 빠르게 심을 수 있겠네요. 하지만 만약에 땅을 파다가 금이 나오거나, 물이 부족해지면 어떻게 해야 할까요? 정지를 하고 다른 일을 해야 할 상황이 생길 수도 있는데 이런 상황에서 어떻게 해야 할까요? 이런 상황에서는 `break`를 사용해서 반복을 종료할 수 있어요.

```python
seeds = 10
gold = True

while seeds > 0:
    print("seed")
    seeds = seeds - 1
    if gold:
        break  # <--- 종료!
```

이 코드는 `gold`가 `True`이면 "seed"가 한 번만 출력되고 종료돼요. 그래서 원래 10번 나와야 하는 문장이 단 한 번만 나오게 돼요.

```python
for i in range(0, 3):
    if i == 1:
        break  # <-- 종료
    print(i)
```

이번에는 `for`에서 `break`를 어떻게 사용하는지 알아봐요. `for` 또한 `break`가 나오면 반복을 중단해요. 그래서 이번에 나온 코드는 i의 값이 1이 될 때 실행을 중단해요.

<span id="continue" />

## 스킵 — continue

`continue`는 반복문의 처음으로 돌아가게 만들어요. `break`는 반복을 중단시키지만, `continue`는 현재 반복을 끝내고 다음 반복으로 넘어가게 돼요.

```python
a = 0
while a < 10:
    a += 1
    if a % 2 == 0:
        continue
    print(a)
```

이번에도 역시 `while`을 먼저 알아봐요. `continue`가 `if` 안에 들어가 있어요. `a % 2`는 a를 2로 나눈 나머지를 알려주는 코드에요. 2로 나눈 나머지는 홀수일 경우 1이 나오고 짝수이면 0이 나와요. 그래서 짝수인 경우에는 `continue`가 실행돼요.

```python
for i in range(10):
    i += 1
    if i % 2 == 0:
        continue
    print(i)
```

이번에는 같은 결과를 주는 프로그램이지만 다른 코드로 만들었어요. `continue`가 `i % 2 == 0`을 만족하는 결과에서만 실행돼요.

## 무한루프

```python
while True:
    print("ping")
```

무한루프를 만들려고 할 때 `while`문의 조건을 입력하는 자리에 True를 넣어요. 이렇게 코드를 만들면 `while`이 무한 반복돼요. 이런 무한루프는 게임을 프로그램할 때 자주 사용되는 코드에요.
