# 튜플 tuple

튜플은 리스트와 아주 비슷해요. 여러 값을 묶어서 담을 수 있거든요. 단, 소괄호 `()`를 써요.

가장 큰 차이는 **한 번 만들면 값을 바꿀 수 없다**는 거예요. 마치 테이프로 봉인한 택배 상자 같아요. 그래서 "변하면 안 되는 값"을 담을 때 좋아요.

```python
# 튜플 만들기 - 소괄호 사용
student = ("Alice", 15, 95.5)
print(student[0])  # Alice
print(student[1])  # 15

# 값을 바꾸려고 하면 오류!
# student[0] = "Bob"  # 에러 발생

# 여러 변수에 한 번에 나눠 담기 (언패킹)
name, age, score = student
print(name)   # Alice
print(age)    # 15
```

## 튜플의 특징

<span id="indexing" />

### 1. 값을 읽는 방법

```python
movie = ("인터스텔라", 2014, "SF")

print(movie[0])   # 인터스텔라   ← 0번부터 시작
print(movie[-1])  # SF          ← 뒤에서 첫 번째
print(movie[0:2]) # ('인터스텔라', 2014)  ← 잘라 읽기
```

리스트처럼 번호(인덱스)로 값을 읽어요. **0번부터** 시작하고, `-1`은 맨 뒤예요. `[0:2]`처럼 잘라 읽기도 돼요.

<span id="plus" />

### 2. 더하기

```python
tuple1 = ("사과", "바나나")
tuple2 = ("딸기", "포도")
print(tuple1 + tuple2)  # ('사과', '바나나', '딸기', '포도')
```

`+`를 쓰면 두 튜플이 이어져서 **새 튜플**이 만들어져요. 원래 튜플은 그대로 남아요.

<span id="repeat" />

### 3. 곱하기

```python
stamp = ("안녕", "반가워")
print(stamp * 3)  # ('안녕', '반가워', '안녕', '반가워', '안녕', '반가워')
```

`*`를 쓰면 내용이 그 숫자만큼 반복돼요. 도장을 여러 번 찍는 것과 같아요.

<span id="len" />

### 4. 길이 구하기

```python
drawer = ("양말", "속옷", "티셔츠", "바지")
print(len(drawer))  # 4
```

`len()`은 튜플 안에 값이 몇 개인지 세어줘요. 리스트, 문자열에도 똑같이 쓸 수 있어요.
