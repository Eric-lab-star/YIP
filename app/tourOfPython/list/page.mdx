# 리스트 list [a, b, c]

지금까지는 숫자 하나, 글자 하나만 다뤘어요. 그런데 값이 여러 개면 어떨까요? 예를 들어 홀수 1, 3, 5, 7, 9를 한꺼번에 담고 싶을 때가 있어요.

이럴 때 쓰는 게 **리스트(list)**예요. 여러 값을 한 번에 담는 **가방**이라고 생각하면 돼요.

<span id="how_to_make" />

## 리스트 만드는 방법

```python
# 리스트 만들기
numbers = [1, 2, 3, 4, 5]
```

대괄호 `[]`가 가방이에요. 그 안에 값을 쉼표 `,`로 구분해서 넣으면 끝이에요. 소풍 가방이라면 `["김밥", "음료수", "과자"]` 처럼요!

<span id="features" />

## 리스트의 특징

### 1. 여러 종류를 담을 수 있어요

```python
any_list = [1, "안녕", True, 3.14]
```

리스트 하나에 숫자, 글자, 불리언을 섞어서 담을 수 있어요. 이름·나이·성적을 한 가방에 넣는 거예요.

### 2. 순서가 있어요

```python
fruits = ["사과", "바나나", "딸기"]
print(fruits[0])   # 사과   ← 0번부터 시작!
print(fruits[1])   # 바나나
print(fruits[-1])  # 딸기   ← 뒤에서 첫 번째
```

리스트의 값에는 번호(인덱스)가 붙어요. **0번부터** 시작해요. `-1`을 쓰면 맨 뒤에서부터 셀 수 있어요.

### 3. 값을 바꿀 수 있어요

```python
fruits = ["사과", "바나나", "딸기"]
fruits[0] = "포도"
print(fruits)  # ['포도', '바나나', '딸기']
```

번호로 위치를 골라 새 값을 넣으면 바뀌어요. 칠판 글씨를 지우고 새로 쓰는 것과 같아요.

### 4. 리스트 안에 리스트도 넣을 수 있어요

```python
matrix = [[1, 2, 3],
          [4, 5, 6],
          [7, 8, 9]]

print(matrix[0][1])  # 2
```

가방 안에 작은 가방을 넣는 거예요. 대괄호를 두 번 쓰면 `matrix[0][1]` 처럼 안쪽 값까지 꺼낼 수 있어요.

### 5. 크기가 자유로워요

```python
fruits = ["사과", "바나나"]

fruits.append("딸기")   # 추가 → ['사과', '바나나', '딸기']
fruits.remove("사과")   # 삭제 → ['바나나', '딸기']
```

언제든 값을 넣고 뺄 수 있어요. 크기를 미리 정하지 않아도 되는 **늘어나는 가방**이에요.

### 6. 잘라낼 수 있어요 (슬라이싱)

```python
numbers = [1, 2, 3, 4, 5]
print(numbers[1:3])  # [2, 3]
print(numbers[:3])   # [1, 2, 3]
print(numbers[2:])   # [3, 4, 5]
```

`리스트[시작:끝]`으로 원하는 구간만 잘라요. 끝 번호는 포함하지 않아요. 시작이나 끝을 비우면 처음부터/끝까지예요.

### 7. 같은 값이 여러 번 들어가도 돼요

```python
numbers = [1, 1, 2, 2, 3, 3]  # 중복 OK! ✅
```

리스트는 똑같은 값을 몇 번이든 담을 수 있어요.

<span id="plus_repeat" />

## 리스트의 연산

리스트는 더하기(`+`)와 곱하기(`*`)를 할 수 있어요. 더하면 두 리스트가 이어지고, 곱하면 그만큼 반복돼요.

### 1. 리스트 더하기

```python
a = [1, 2, 3]
b = [4, 5, 6]
print(a + b)  # [1, 2, 3, 4, 5, 6]
```

`+`를 쓰면 두 리스트가 순서대로 이어져요. 원래 리스트 `a`, `b`는 그대로 남아요.

### 2. 리스트 반복하기

```python
a = [0] * 5
print(a)  # [0, 0, 0, 0, 0]
```

`*`를 쓰면 내용이 그 숫자만큼 반복돼요. `[0] * 5`처럼 초기값을 한 번에 만들 때 편해요.

### 3. 길이 구하기

```python
numbers = [1, 2, 3, 4]
print(len(numbers))  # 4
```

`len()`은 리스트 안에 값이 몇 개인지 세어줘요.

<span id="edit" />

## 리스트의 수정과 삭제

- **수정**: `fruits[1] = "포도"` 처럼 번호로 위치를 골라 새 값을 넣어요.
- **삭제**: `remove(값)`은 값으로 지우고, `del 리스트[번호]`는 번호로 지워요.

```python
fruits = ['사과', '바나나', '딸기']
fruits[1] = '포도'        # 수정 → ['사과', '포도', '딸기']

fruits.remove('사과')     # 값으로 삭제 → ['포도', '딸기']
del fruits[0]             # 번호로 삭제 → ['딸기']
```

슬라이싱으로 여러 개를 한 번에 바꿀 수도 있어요.

```python
ive = ['장원영', '이서', '레이']
ive[0:2] = ['안유진', '리즈']
print(ive)  # ['안유진', '리즈', '레이']
```

## 리스트 메소드

리스트에는 편리한 기능(메소드)이 미리 들어 있어요. 하나씩 볼게요.

<span id="append" />

### 1. 덧붙이기 append()

```python
colors = ['red', 'orange', 'yellow']
colors.append('purple')
print(colors)  # ['red', 'orange', 'yellow', 'purple']
```

`append()`는 **맨 끝에** 값을 하나 추가해요.

<span id="sort" />

### 2. 정렬하기 sort()

```python
numbers = [100, 19, 98, 32, 6, 1]
numbers.sort()
print(numbers)  # [1, 6, 19, 32, 98, 100]
```

`sort()`는 작은 값부터 큰 값 순서로 정렬해요. 큰 순서로 하려면 `sort(reverse=True)`를 써요.

<span id="reverse" />

### 3. 뒤집기 reverse()

```python
numbers = [1, 2, 3, 4, 5]
numbers.reverse()
print(numbers)  # [5, 4, 3, 2, 1]
```

`reverse()`는 순서를 거꾸로 뒤집어요.

<span id="index" />

### 4. 번호 알아내기 index()

```python
fruits = ["사과", "바나나", "딸기"]
print(fruits.index("딸기"))  # 2
```

`index()`는 그 값이 **몇 번째**에 있는지 알려줘요.

<span id="insert" />

### 5. 삽입하기 insert()

```python
fruits = ["사과", "바나나", "딸기"]
fruits.insert(1, "포도")
print(fruits)  # ['사과', '포도', '바나나', '딸기']
```

`insert(번호, 값)`은 **원하는 위치**에 값을 끼워 넣어요. 뒤의 값들은 한 칸씩 밀려요.

<span id="remove" />

### 6. 제거하기 remove()

```python
numbers = [100, 0, 1, 100]
numbers.remove(100)
print(numbers)  # [0, 1, 100]
```

`remove(값)`은 그 값을 찾아 지워요. 같은 값이 여러 개면 **맨 앞의 하나만** 지워요.

<span id="pop" />

### 7. 꺼내기 pop()

```python
numbers = [100, 0, 1]
last = numbers.pop()
print(last)     # 1   ← 꺼낸 값을 돌려줌
print(numbers)  # [100, 0]
```

`pop()`은 값을 **지우면서 그 값을 돌려줘요**. 이게 `remove`·`del`과 다른 점이에요.

<span id="count" />

### 8. 개수 세기 count()

```python
numbers = [100, 0, 1, 100]
print(numbers.count(100))  # 2
```

`count(값)`은 그 값이 리스트에 **몇 번** 들어있는지 세어줘요.
