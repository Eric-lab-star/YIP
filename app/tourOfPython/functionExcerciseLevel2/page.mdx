## 2. 매개변수가 있는 함수 만들기

매개변수가 있는 함수 만들기 연습을 해봅시다. 아래의 코드는 예시입니다.

> 복사 붙여넣기 사용하지 말고 손으로 한 땀 한 땀 만드세요.

#### 1. 더하기 함수

```python
def add(x, y):
    print(x + y)
```

#### 2. 제목 함수

```python
def title(width, text):
    print(text.strip().center(width, "="))
```

#### 3. 사용자 만들기 함수

```python
def createUser(id, name):
    print(f"id: {id}, name: {name}")
```

#### 4. 아이스 음료 만들기 함수

```python
def ice_maker(drink):
    print(f"ice {drink}")
```

#### 5. 하트 생성기 함수

```python
def heart_maker(n):
    print(" ❤️ " * n)
```

#### 6. 나무 만들기 함수

```python
def createTree(num):
    for i in range(num):
        print((" * " * (i + 1)).center(40, " "))
```

## 3. 값을 반환하는 함수 만들기

값을 반환하는 함수 만들기 연습을 해봅시다.

> 복사 붙여넣기 사용하지 말고 손으로 한 땀 한 땀 만드세요.

#### 1. 더하기 함수

```python
def add(x, y):
    return x + y

result = add(5, 3)
```

#### 2. 로그인 함수

```python
def loggedIn(user):
    if user.state:
        return True
    else:
        return False
```

#### 3. 별 만들기 함수

```python
def createStar(num):
    return "*" * num
```

#### 4. 평균 구하기 함수

```python
def average(x):
    s = sum(range(x + 1))
    print(s / x)
```
