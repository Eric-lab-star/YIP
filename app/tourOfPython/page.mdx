## 🐍 파이썬이란 무엇인가

파이썬은 컴퓨터에게 말을 걸 수 있는 쉬운 언어예요. 우리가 한국어로 이야기하듯이, 컴퓨터와는 파이썬 말로 이야기해요. 일상생활로 비유하면 리모컨을 생각해 보세요. 버튼을 누르면 TV가 켜지고, 채널이 바뀌죠? 파이썬은 컴퓨터 리모컨이에요. 우리가 버튼, 즉 코드를 입력하면 컴퓨터가 그대로 행동해요!

- 📖 글처럼 읽혀서 어렵지 않아요
- ✨ 짧게 써도 많은 일을 해요
- 🐤 처음 코딩 배우는 사람에게 딱 좋아요

## 파이썬 예제

```python
# 파이썬으로 피보나치 수열의 n번째 숫자 구하기
def fibonacci(n):
    a = 0
    b = 1

    # Check if n is less than 0
    if n < 0:
        print("Incorrect input")

    # Check if n is equal to 0
    elif n == 0:
        return 0

    # Check if n is equal to 1
    elif n == 1:
        return b
    else:
        for i in range(1, n):
            c = a + b
            a = b
            b = c
        return b

print(fibonacci(9))
```

### 웹 서버 — Django

![django logo](https://r2.kimkyungsub.com/django.png)

Django는 파이썬으로 웹 사이트를 빠르고 쉽게 만들 수 있게 도와주는 웹 프레임워크에요. 웹 사이트를 집이라고 생각해보면, 파이썬만 이용하면 벽돌을 하나하나 직접 쌓아서 집을 짓는 것과 같아요. 하지만 Django를 사용하면 이미 설계된 집에서 인테리어를 바꾸는 것이라고 생각할 수 있어요. 그래서 개발자는 핵심기능에만 집중할 수 있어요.

| 특징 | 설명 |
| --- | --- |
| 관리자 패널 | 자동으로 관리자 페이지 생성 |
| ORM | SQL 대신 python 코드로 DB 제어 |
| 인증 시스템 | 로그인, 회원가입 기능 내장 |
| 보안 | CSRF, XSS 등 기본 보안 |

**사용하는 기업들**

### 게임 개발 — pygame

![pygame logo](https://r2.kimkyungsub.com/pygame_logo.png)

Pygame은 Python으로 게임을 만들 수 있게 도와주는 라이브러리예요. 게임을 만드는 것을 애니메이션 만들기로 비유를 하면, 파이썬만 사용하는 것은 흰 종이에 그림을 하나하나 직접 그리고, 움직임도 직접 계산하는 것과 같아요. Pygame을 사용하면 필요한 도구가 모두 준비된 상태에서 그림을 그리는 것이에요.

| 특징 | 설명 |
| --- | --- |
| 화면 | 게임 창 생성 및 관리 |
| 그리기 | 사각형, 원, 선 등 도형 그리기 |
| 이미지 | 스프라이트 이미지 불러오기 |
| 입력 | 키보드, 마우스 입력 감지 |
| 사운드 | 배경음악, 효과음 재생 |
| 시간 | FPS 조절 |

### 하드웨어 제어 — micro python

![micropython logo](https://r2.kimkyungsub.com/micropython_logo.png)

마이크로 파이썬은 마이크로컨트롤러 위에서 동작하도록 만들어진 초경량 파이썬이에요. 일반 파이썬과 마이크로 파이썬의 차이를 자동차로 비유하면, 일반 파이썬은 고속도로를 달리는 일반 승용차이고 마이크로 파이썬은 좁은 골목을 누비는 오토바이라고 생각할 수 있어요. 문법은 같지만 아주 작은 하드웨어에서도 돌아갈 수 있도록 가볍게 최적화된 버전이에요.

| 특징 | 설명 |
| --- | --- |
| 문법 | 배운 파이썬 그대로 사용가능 |
| 크기 | 초경량. 256KB RAM에서도 동작 |
| 하드웨어 직접 제어 | LED, 센서, 모터 등 직접 조작 |

[마이크로 파이썬 더 알아보기 🔗](https://micropython.org/)
