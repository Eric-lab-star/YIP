# 파이썬 Editor 소개

그림을 그린다고 하면 어디에다가 그리고 싶은가요? 모든 준비물과 시설이 잘 갖추어진 미술실에서 그리고 싶은가요? 아니면 아무것도 없는 땅바닥에서 그림을 그리고 싶은가요? 아마 대부분의 사람들은 여러 가지로 편리하고 쾌적한 미술실에서 그림을 그리고 싶다고 생각했을 거예요. IDE도 마찬가지에요. IDE란 코딩을 위한 미술실이에요. 코딩을 위한 각종 편의 기능을 지원하고 개발자들은 이런 편의 기능 덕분에 더 빠르게 작업을 할 수 있어요.

## IDE를 사용하면 좋은 점들

1. 자동완성
2. 오류표시
3. 색깔구분
4. 바로실행

## 대표적인 IDE

### 1. vscode

![vscode 홈페이지](https://r2.kimkyungsub.com/vscodeHome.png)

| 특징 | 설명 |
| --- | --- |
| 무료 | Microsoft가 만들었지만 완전 무료이고, 소스코드도 공개되어 있어요. |
| 가볍고 빠름 | 무거운 IDE(IntelliJ, Eclipse 등)와 달리, 텍스트 에디터 기반이라 실행이 빠릅니다. |
| 확장 프로그램 | 기본 기능은 단순하지만, 마켓플레이스에서 확장 프로그램을 설치해 원하는 기능을 추가할 수 있어요. |
| 실시간 협업 가능 | 다른 사람과 같은 코드를 동시에 편집하는 기능도 지원해요. |

### 2. pycharm

![pycharm 홈페이지](https://r2.kimkyungsub.com/pycharmHome.png)

| 특징 | 설명 |
| --- | --- |
| 강력한 분석 기능 | 타입 추론이 정확해서 자동완성 품질이 높음 |
| 강력한 분석 기능 | 변수명 하나를 바꾸면 프로젝트 전체에서 관련된 곳을 자동으로 수정해줘요. |
| 강력한 디버깅 기능 | 코드를 실행하면서 중간에 멈추고, 변수값을 실시간으로 확인할 수 있어요. |

## VSCode으로 파이썬 코드 실행하기

먼저 vscode가 설치되었는지 확인 후 진행하세요.

### 새폴더 만들기

![vscode에서 새 폴더 만들기 1](https://r2.kimkyungsub.com/vscode/vscode_howto1.png)

1. 바탕화면에서 마우스 우클릭을 하면 새로만들기에서 폴더를 클릭해서 새 폴더를 만들 수 있어요.

---

![vscode에서 폴더 이름 바꾸기 2](https://r2.kimkyungsub.com/vscode/vscode_howto2.png)

2. 폴더 이름은 본인의 이름으로 바꾸세요.

---

![helloworld 폴더 만들기 3](https://r2.kimkyungsub.com/vscode/vscode_howto3.png)

3. 본인 이름의 폴더 안에서 helloworld라는 폴더를 만드세요.

---

![Code로 열기 4](https://r2.kimkyungsub.com/vscode/vscode_howto4.png)

4. helloworld 폴더에 마우스 커서를 두고 우클릭을 하면 보이는 "Code로 열기"를 클릭하세요.

---

![신뢰 확인 창 5](https://r2.kimkyungsub.com/vscode/vscode_howto5.png)

5. 위의 그림과 같은 창이 나오면 파란색 버튼을 누르세요.

---

![새 파일 만들기 6](https://r2.kimkyungsub.com/vscode/vscode_howto6.png)

6. vscode가 실행되면 왼쪽 사이드 바에서 파일 아이콘을 클릭해서 새로운 파일을 만드세요.

---

![파일 이름 main.py 7](https://r2.kimkyungsub.com/vscode/vscode_howto7.png)

7. 파일 이름을 main.py로 합니다.

---

![코드 작성하기 8](https://r2.kimkyungsub.com/vscode/vscode_howto8.png)

8. 파일을 클릭하면 이곳에 코드를 작성할 수 있어요.

---

![코드 실행하기 9](https://r2.kimkyungsub.com/vscode/vscode_howto9.png)

9. 오른쪽에 보이는 세모 모양의 재생 버튼을 클릭하면 코드가 실행돼요.

---

![실행 결과 확인 10](https://r2.kimkyungsub.com/vscode/vscode_howto10.png)

10. 하단에 보면 이렇게 실행되는 창이 나와요.

---
